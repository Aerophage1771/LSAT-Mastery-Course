# Module 37: Passage Function -- Website Content Improvement Report

**Scope:** The 2 website lessons (37-1 through 37-2) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 37 covers Passage Function questions -- "big picture" purpose questions that ask why the author wrote the passage. This is the capstone of the Function hierarchy (M35 Phrase → M36 Paragraph → M37 Passage). The content is well-organized and correctly distinguishes this type from Main Point questions, but shares the persistent no-practice gap:

1. **Zero practice questions** -- neither lesson includes a question-card or complete worked example.
2. **The Main Point vs. Purpose distinction is the module's most valuable insight** -- 37-2's tie-breaker correctly explains that the main point is what the author concludes and the purpose is why they wrote it. This distinction is critical and well-articulated.
3. **Three stem categories are well-differentiated** -- Direct Inquiry, Comparative Inquiry, and Question-Based Inquiry cover the full range of Passage Function stems.
4. **The five-item purpose vocabulary is strong** -- arguing for change, explaining a phenomenon, evaluating a theory, reconciling viewpoints, correcting a misconception covers the most common passage purposes.
5. **Good scope awareness** -- 37-2 explicitly warns against Too Narrow, Too Broad, and Wrong Verb answers, which is the right diagnostic framework.

## Lesson-by-Lesson Review

### Lesson 37-1: Germaine's Guide to RC Passage Function Questions (~15 blocks)

**Role:** Module introduction, concept overview, and stem identification guide.

**Strengths:**
- Clear opening that correctly frames the task as identifying the author's overarching intent
- The five-item purpose vocabulary (argue, explain, evaluate, reconcile, correct) gives students a concrete set of verbs to anchor predictions
- Three stem categories (Direct Inquiry, Comparative Inquiry, Question-Based Inquiry) are well-differentiated and cover the full range
- The Question-Based Inquiry category is unique to this module and insightful -- stems that rephrase purpose as a question the passage answers
- Frequency data (100 appearances, ~5%) correctly positions this as a regular question type
- The Overarching Takeaway callout correctly emphasizes scope, breadth, and intent

**Issues:**
- **No practice question.** The purpose vocabulary and stem categories are introduced but never tested.
- **No comparison to Main Point questions.** Since Passage Function and Main Point are closely related and often confused, an explicit comparison would be highly valuable.
- **No comparison to Paragraph Function (M36).** The module does not acknowledge its position in the Function hierarchy.
- **The Comparative Inquiry category needs more guidance.** Comparative Passage Function questions are complex (involving two passages), and the category description does not address the additional challenge of synthesizing two purposes.

**Suggested edits:**
- Add a "Passage Function vs. Main Point" comparison table showing how the same passage could generate both question types.
- Add a brief Function Hierarchy note connecting M35 → M36 → M37.
- Expand the Comparative Inquiry section with a brief note about the additional complexity of dual-passage purpose questions.
- Add a practice exercise asking students to match passages to purpose verbs.

---

### Lesson 37-2: Step-by-Step Approach (~15 blocks)

**Role:** The complete method for solving Passage Function questions.

**Strengths:**
- Clean three-step method: Synthesize Main Point + Structure, Predict Primary Goal, Test Against Scope
- Step 1's three-question framework (main conclusion, argument building method, overall tone) is comprehensive and systematic
- Step 2's emphasis on starting with a "strong, abstract verb" is excellent advice that focuses the prediction
- The worked prediction example ("critique or refute a widely held view") demonstrates Step 2 concretely
- Elimination criteria (Too Narrow, Too Broad, Wrong Verb) are precise and actionable
- Two tie-breakers (Main Point Connection, Tone Check) address the two most common decision points

**Issues:**
- **No question-card.** The method is strong but entirely theoretical.
- **No worked example beyond the brief prediction fragment.** The full 3-step process is never demonstrated on a complete question.
- **The Main Point Connection tie-breaker is the module's best insight but needs elaboration.** The idea that "main point is what the author concludes, purpose is why they wrote" deserves a concrete example showing both questions answered for the same passage.
- **No warning about the "too descriptive" trap.** Students sometimes select answers that describe the passage's topic rather than its purpose (e.g., "discusses environmental policy" instead of "argues for stricter regulations").

**Suggested edits:**
- Add a complete worked example demonstrating all three steps on a passage outline.
- Add a question-card after the method section.
- Expand the Main Point Connection tie-breaker with a concrete dual-answer example.
- Add a "too descriptive" trap warning: answers that describe the topic rather than the author's purpose.

## Structural Issues

### 1. No Practice Questions

Both lessons are pure instruction. For a question type appearing 100 times, this is a notable gap.

**Recommendation:** Add at least one question-card to 37-2.

### 2. Missing Main Point Comparison

Passage Function and Main Point are the two most commonly confused RC question types. The module does not address this overlap.

**Recommendation:** Add an explicit comparison to 37-1 showing how the same passage generates different Passage Function and Main Point questions.

### 3. Incomplete Function Hierarchy

Modules 35, 36, and 37 form a natural hierarchy, but none explicitly links to the others.

**Recommendation:** Add a brief hierarchy note to 37-1 showing M37 as the capstone of the Function progression.
