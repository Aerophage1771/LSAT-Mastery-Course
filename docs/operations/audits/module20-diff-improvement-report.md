# Module 20: Agree/Disagree (Point at Issue) -- Website Content Improvement Report

**Scope:** The 10 website lessons (20-1 through 20-10) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 20 is one of the strongest modules in the course. The topic naturally divides into clean sub-types (four disagreement patterns, two agreement patterns), and the lessons exploit this structure well. Every drill lesson (20-3 through 20-5, 20-7 through 20-9) includes a real PT question with full five-choice wrong-answer analysis. The Reference Guide (20-10) is comprehensive and well-organized. The primary improvement opportunities are:

1. **No "try it first" prompts before any worked analysis.** This is the single highest-priority gap. Six lessons include question-cards followed immediately by full analysis with no pause for active recall.
2. **Lesson 20-6 (Advanced Strategy) has a practice question setup but the question-card itself is missing.** The analysis references Martinez/Chandra and answer choices (A)-(E), but there is no stimulus, question stem, or answer options displayed to the student.
3. **Lesson 20-1 (Introduction) has no practice element at all.** The student reads conceptual overview material without ever attempting anything.
4. **The Reference Guide (20-10) is excellent but missing frequency data.** Unlike other modules that include how often the question type appears, Module 20's reference guide has no frequency/prevalence information.
5. **Minor module-number mismatch.** The heading says "Module 20" consistently across lessons, which is correct for this evaluation.

## Lesson-by-Lesson Review

### Lesson 20-1: Introduction (14 blocks)

**Role:** Module introduction and conceptual overview.

**Strengths:**
- Clear, concise Why This Matters section that immediately explains what makes this question type unique (two speakers)
- Question Stem recognition table with three stem patterns is practical and actionable
- The Overlap Test is introduced early and explained clearly with the YES/NO/Unknown framework
- Common Dialogue Structures list (4 patterns) gives students a useful mental catalog
- Concept Snapshot (Marta/Niko library example) is simple and well-chosen
- Key Principle callout is the single most important takeaway: silence is not a position

**Issues:**
- **No practice element.** The student reads 14 blocks of pure theory without being asked to try anything. Even a mini-exercise ("Which of these would be the point at issue?") would improve retention.
- **The Common Dialogue Structures list previews patterns taught in later lessons** but does not cross-reference which lessons cover which patterns. A forward reference ("Lesson 3 covers Direct Contradiction; Lesson 4 covers Different Explanations") would help navigation.
- **Explicit vs. Implicit commitments are not mentioned.** This is a critical concept for harder questions. Lesson 20-2 touches on it, but the introduction should at least flag that not all commitments are stated outright.

**Suggested edits:**
- Add a mini-exercise after the Concept Snapshot: present a short two-speaker dialogue and ask the student to identify the point of disagreement before revealing the answer.
- Add a brief mention of explicit vs. implicit commitments as a preview of what makes harder questions harder.
- Cross-reference the Common Dialogue Structures to their corresponding drill lessons.

### Lesson 20-2: Step-by-Step Guide (20 blocks)

**Role:** The complete method for solving Agree/Disagree questions.

**Strengths:**
- Clean four-step method: Map Positions, Identify Committed Claims, Apply Overlap Test, Verify Both Commit
- Step 1 includes an ordered list for decomposing each speaker (Conclusion, Premises, Concessions) -- practical and specific
- The Overlap Test table (Disagreement vs. Agreement target patterns) is clear and scannable
- Worked Method Snapshot (Chen/Dara recycling) walks through the method coherently
- Wrong Answer Patterns section identifies three key traps with clear descriptions
- Timing Tip callout is actionable (extra 15-20 seconds up front)

**Issues:**
- **No "try it first" prompt.** The Worked Method Snapshot is pure lecture -- the student is told the answer without being asked to attempt it first. This is a missed active-learning opportunity.
- **The method is called "four-step" but the website version has no Step 2 independent of Step 3.** Step 2 (Identify Committed Claims) and Step 3 (Apply Overlap Test) are both about commitment. The distinction is valid but could be sharper -- Step 2 is about building the claim list, Step 3 is about comparing them across speakers.
- **Wrong Answer Patterns section has only 3 items.** The Reference Guide (20-10) has 7 traps. At minimum, the Step-by-Step should mention the most dangerous traps beyond the top 3.
- **No question-card.** This is the method lesson, so a practice question after the worked snapshot would let students test the method immediately.

**Suggested edits:**
- Add a "Try this" prompt before the Worked Method Snapshot.
- Add a question-card after the snapshot for independent practice.
- Consider previewing 1-2 more trap patterns (Scope Mismatch, Mischaracterization) since these appear frequently on harder questions.

## Structural Issues

### 1. Missing "Try It First" Prompts (HIGH PRIORITY)

Six lessons (20-3 through 20-5, 20-7 through 20-9) include question-cards followed immediately by full worked analysis. None includes a prompt asking the student to attempt the question independently before reading the solution. This is the single highest-impact improvement available: adding one line before each analysis section would convert passive reading into active practice.

### 2. Missing Question-Card in Lesson 20-6

The Advanced Strategy Guide references a Martinez/Chandra electric bus dialogue and provides a complete five-choice analysis with badges and trap labels -- but the actual question-card (stimulus text, stem, and answer options) is never displayed. A student encounters the analysis without having seen the question. This needs to be fixed before the lesson functions as intended.

### 3. Analysis Format Inconsistency

Lesson 20-2 teaches a four-step method (Map Positions, Identify Claims, Apply Overlap Test, Verify Commitment). The drill lessons (20-3 through 20-5, 20-7 through 20-9) use a different three-step format (Deconstruction, Abstraction, Evaluation). Both formats are pedagogically sound, but the switch between them is never acknowledged. A brief note in the first drill ("This analysis uses a condensed three-step format") would prevent confusion.

### 4. Definitional Rule vs. Definitional Clash Boundary

Lessons 20-3 and 20-8 cover related but distinct patterns. The Reference Guide (20-10) distinguishes them well in the table, but the drill lessons themselves do not explicitly compare the two. A brief note in Lesson 20-8 contrasting it with Lesson 20-3 would help students who struggle with the distinction.
