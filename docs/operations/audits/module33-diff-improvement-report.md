# Module 33: Passage Principle -- Website Content Improvement Report

**Scope:** The 3 website lessons (33-1 through 33-3) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 33 covers Passage Principle questions across three compact lessons: an introduction with stem identification, a step-by-step approach, and a pattern recognition guide. The content is accurate, well-organized, and the stem identification breakdown in 33-1 is particularly strong. However, the module has significant gaps that limit its standalone effectiveness:

1. **Zero practice questions across all three lessons** -- the student reads the entire module (~25 blocks total) without encountering a single question-card or worked example with actual LSAT stimulus text. This is the module's most critical weakness.
2. **Lesson 33-3 is too thin** -- at only one breakdown block, it provides three pattern types but no examples, no exercises, and no depth. It reads more like a sidebar than a full lesson.
3. **No worked examples anywhere** -- none of the three lessons include a concrete LSAT-style stimulus with step-by-step analysis. The method lesson (33-2) uses hypothetical fragments but never walks through a complete question.
4. **Strong stem identification** -- the 33-1 breakdown covering Direct Principle, Generalization/Criterion, and Comparative Principle phrasing is comprehensive and well-categorized.

---

## Lesson-by-Lesson Review

### Lesson 33-1: Germaine's Guide to RC Passage Principle Questions (~14 blocks)

**Role:** Module introduction, concept overview, and stem identification guide.

**Strengths:**
- Clear opening that defines the core task: moving from specific examples to general rules
- The three-step framework (identify scenario, isolate logic, generalize) is actionable
- The stem identification breakdown is excellent -- three categories (Direct Principle, Generalization/Criterion, Comparative Principle) with real stem language and color-coded badges
- The Overarching Takeaway callout provides a strong, memorable summary
- The frequency note (35 appearances) gives students useful context about question rarity

**Issues:**
- **No practice question.** The lesson introduces the concept and stem identification but never asks the student to try anything. Even a single "identify which category this stem belongs to" exercise would improve engagement.
- **The three-step framework is generic.** "Identify the scenario, isolate the logic, generalize" could describe almost any LSAT question type. The lesson would benefit from a concrete passage excerpt showing each step in action.
- **No trap answer patterns.** The lesson describes correct answer characteristics but does not warn about common wrong-answer types (e.g., principles that are too narrow, too broad, or that reverse the direction of the reasoning).

**Suggested edits:**
- Add a brief worked example after the three-step framework using a short hypothetical passage excerpt.
- Add a "Common Traps" callout listing the 2-3 most frequent wrong-answer patterns for this question type.
- Add a mini-exercise after the stem identification breakdown: present 2-3 stems and ask students to categorize them.

---

### Lesson 33-2: Step-by-Step Approach (~14 blocks)

**Role:** The complete method for solving Passage Principle questions.

**Strengths:**
- Clear three-step method: Isolate the Specific Argument, Formulate the Underlying Rule, Test Answer Choices
- Each step includes a running example (court ruling / advertisement), which helps illustrate the process
- The elimination criteria (Too Specific, Too Broad, Mismatch) are practical and actionable
- Two tie-breaker strategies (Justification Test, Focus on the Difference) are sophisticated and useful for students already at an intermediate level

**Issues:**
- **No question-card.** This is the method lesson but includes no real LSAT question for practice. The running example is a hypothetical fragment, not a question-card the student can attempt before seeing the analysis.
- **The running example is incomplete.** It shows the court ruling scenario across Steps 1-2 but Step 3 does not show actual answer choices being evaluated against the rule. The student sees elimination criteria in the abstract but never sees them applied.
- **The tie-breakers assume two final contenders** but never show how to get there. A brief demonstration of narrowing from 5 to 2 would make the advice more concrete.

**Suggested edits:**
- Add a complete worked example with actual answer choices (or realistic simulated ones) showing the full 3-step process including elimination.
- Add a question-card after the method walkthrough so students can practice independently.

---

### Lesson 33-3: Pattern Recognition (~4 blocks)

**Role:** Pattern taxonomy for Passage Principle questions.

**Strengths:**
- Identifies three useful patterns: Standard of Evaluation, Justification for Action, Underlying Logical Rule
- The pattern descriptions are clear and each includes the "clue" (what triggers the pattern) and the "structure" (what the correct answer looks like)
- Badge colors provide quick visual differentiation

**Issues:**
- **Extremely thin.** This lesson is a single breakdown block with three items. At ~4 blocks, it is the shortest lesson in any RC module and feels incomplete.
- **No examples for any pattern.** Each pattern describes its structure abstractly but does not include a concrete passage excerpt or question demonstrating the pattern in action.
- **No exercises.** The student cannot practice pattern recognition because there are no stimuli to categorize.
- **No connection to the method.** The patterns are not linked back to the 3-step approach from 33-2. A note explaining which step benefits most from pattern recognition would help integrate the lessons.

**Suggested edits:**
- Add at least one short worked example per pattern showing a passage excerpt + the principle that matches.
- Add a brief categorization exercise: present 3 short scenarios and ask students to identify which pattern applies.
- Add a transition paragraph linking these patterns to the Step 2 formulation process from 33-2.
- Consider merging this lesson into 33-2 as a "Pattern Recognition" section, since it is too short to stand alone.

---

## Structural Issues

### 1. No Practice Questions in the Entire Module

All three lessons are pure theory. The student reads about how to identify Passage Principle stems, how to apply the 3-step method, and how to recognize patterns -- but never actually practices any of these skills. This is the most critical gap.

**Recommendation:** Add at least one question-card to 33-2 (the method lesson) and consider adding a categorization exercise to 33-1 or 33-3.

### 2. Lesson 33-3 Is Too Thin to Be a Standalone Lesson

At ~4 blocks, 33-3 is a sidebar, not a lesson. It would work better as a section within 33-2.

**Recommendation:** Either expand 33-3 with examples and exercises, or merge its content into 33-2 as a "Pattern Recognition" subsection.

### 3. No Worked Examples with Complete Analysis

None of the three lessons walk through a complete question from stimulus to correct answer. The court ruling example in 33-2 is a useful fragment but stops short of demonstrating the full evaluation-and-elimination process.

**Recommendation:** Add one complete worked example to 33-2 that includes stimulus, question stem, all five answer choices, and step-by-step analysis.
