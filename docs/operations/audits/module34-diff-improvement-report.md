# Module 34: Phrase Meaning -- Website Content Improvement Report

**Scope:** The 2 website lessons (34-1 through 34-2) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 34 covers Phrase Meaning questions in two lessons: an introduction with stem identification, and a step-by-step approach with tie-breakers. The content is well-organized and pedagogically sound. The stem identification in 34-1 is the strongest in the RC section, covering four distinct stem categories (Direct Definition, Contextual Substitution, Rationale for Word Choice, Exemplification). However, the module shares the same critical gap as its neighbors:

1. **Zero practice questions** -- neither lesson includes a question-card or a complete worked example with actual passage text and answer choices.
2. **The substitution method is strong but untested** -- 34-2's Step 3 (substitution technique) is an excellent tactical tool but the student never gets to apply it against real answer choices.
3. **Four stem categories is unusually rich** -- at 106 appearances, Phrase Meaning is one of the higher-frequency RC question types, and the four-category breakdown in 34-1 reflects this. This is a genuine strength.
4. **The "qualified approval" example is effective** -- the running example in 34-2 demonstrates the substitution method well, but it needs to be extended into a full worked question.

## Lesson-by-Lesson Review

### Lesson 34-1: Germaine's Guide to RC Phrase Meaning Questions (~15 blocks)

**Role:** Module introduction, concept overview, and stem identification guide.

**Strengths:**
- Clear opening that correctly frames the task as "textual detective" work rather than vocabulary testing
- The four-item skill list (examine surrounding sentences, substitute, consider tone, identify specialized usage) is practical and complete
- The stem identification breakdown is the richest in the RC modules -- four categories (Direct Definition, Contextual Substitution, Rationale for Word Choice, Exemplification) cover the full range of how LSAT tests phrase meaning
- The Overarching Takeaway callout correctly emphasizes context-specific meaning over dictionary definitions
- Frequency note (106 appearances, ~5%) gives useful weight to the topic

**Issues:**
- **No practice question.** Despite the rich stem categorization, students never practice identifying stem types or applying the framework.
- **The "common word in a specialized way" bullet is important but buried** in the introductory list. This is one of the LSAT's favorite tricks and deserves more emphasis -- perhaps a callout or a dedicated example.
- **No trap answer patterns.** The lesson does not warn about the most common wrong-answer type: the dictionary-definition choice that ignores context.

**Suggested edits:**
- Add a "Common Trap" callout about dictionary-definition answer choices that ignore the passage's specific context.
- Promote the "specialized/metaphorical usage" concept to a standalone example or callout.
- Add a brief stem categorization exercise.

---

### Lesson 34-2: Step-by-Step Approach (~13 blocks)

**Role:** The complete method for solving Phrase Meaning questions.

**Strengths:**
- Clean three-step method: Locate + Read Context, Predict Based on Role, Test by Substitution
- Step 2's framing ("What job is this word doing here?") is an excellent reframe that shifts students from vocabulary to function
- The "qualified approval" example effectively demonstrates the prediction process
- Two tie-breakers (Tone and Scope Check, "Why This Word?" Test) are sophisticated and address real decision points
- The "Why This Word?" tie-breaker is particularly strong -- it teaches students to think about authorial intent

**Issues:**
- **No question-card.** The method is explained well but never demonstrated on a complete question with answer choices.
- **The "qualified approval" example stops at prediction.** It shows how to predict meaning but does not follow through with actual answer choices to test against.
- **No wrong-answer analysis.** The lesson describes the substitution technique but never shows a wrong answer being eliminated through substitution.

**Suggested edits:**
- Extend the "qualified approval" example into a full worked question with 5 answer choices showing the substitution technique in action.
- Add a question-card after the method walkthrough.
- Add a brief section showing one wrong answer being eliminated via substitution.

---

## Structural Issues

### 1. No Practice in Either Lesson

Both lessons are pure instruction with no interactive practice. For a question type that appears 106 times, this is a notable gap.

**Recommendation:** Add at least one question-card to 34-2.

### 2. Strong Method but No Demonstration

The substitution method (34-2, Step 3) is one of the most concrete and actionable techniques in the RC modules, but it is never demonstrated on a real question. This is a missed opportunity.

**Recommendation:** Add a complete worked example showing substitution applied to each answer choice in turn.
