# Module 36: Paragraph Function -- Website Content Improvement Report

**Scope:** The 2 website lessons (36-1 through 36-2) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 36 covers Paragraph Function questions across two lessons. This module sits in a clear hierarchy with Phrase Function (M35) and Passage Function (M37), and its content correctly scales up the analytical lens from individual phrases to entire paragraphs. The lessons are well-structured and pedagogically sound, but share the persistent gap of zero practice. Key findings:

1. **Zero practice questions** -- neither lesson includes a question-card or complete worked example.
2. **The "bridge" metaphor in 36-2 is excellent** -- framing the paragraph as a logical bridge between its neighbors is a powerful and memorable teaching device.
3. **Only one stem category in 36-1** -- unlike M34 (four categories) and M35 (three categories), M36 has only "Direct Inquiry About Role." This is likely accurate (Paragraph Function stems are relatively uniform), but a note acknowledging this would prevent students from thinking the lesson is incomplete.
4. **Strong function vocabulary** -- the five-item function list in 36-1 (introducing a problem, providing evidence, presenting an alternative view, evaluating a proposal, summarizing a debate) gives students a practical toolkit.
5. **Good verb-based tie-breaker** -- the "Focus on the Verbs" technique in 36-2 teaches precision that transfers to other question types.

## Lesson-by-Lesson Review

### Lesson 36-1: Germaine's Guide to RC Paragraph Function Questions (~12 blocks)

**Role:** Module introduction, concept overview, and stem identification guide.

**Strengths:**
- Clear opening that correctly positions the task: identify the paragraph's role within the passage structure
- The five-item function list provides a comprehensive toolkit covering the most common paragraph roles
- The stem identification breakdown, while only one category, includes diverse stem examples that cover the range
- The Overarching Takeaway callout correctly emphasizes abstract functional language over content summary
- Frequency data (66 appearances) and cross-references to Structure and Inference questions establish context

**Issues:**
- **Only one stem category.** While this may accurately reflect the uniformity of Paragraph Function stems, students who have seen 3-4 categories in other modules may find this surprising. A brief note explaining why would help.
- **No practice question.** The function vocabulary is introduced but never tested.
- **No comparison to Phrase Function (M35) or Passage Function (M37).** Since these three modules form a natural hierarchy (phrase → paragraph → passage), an explicit comparison showing how the analytical lens scales would help students see the connection.
- **The function list is useful but static.** It lists five roles without explaining which are most common or how to quickly identify them.

**Suggested edits:**
- Add a note explaining the single stem category: "Paragraph Function stems are relatively uniform in phrasing, almost always asking directly about the 'function,' 'purpose,' or what a paragraph 'does.'"
- Add a brief Function Hierarchy comparison: Phrase Function → Paragraph Function → Passage Function, showing how the same analytical skill applies at different scales.
- Add frequency or prominence data for the five function types if available.

---

### Lesson 36-2: Step-by-Step Approach (~14 blocks)

**Role:** The complete method for solving Paragraph Function questions.

**Strengths:**
- Clean three-step method: Summarize Content, Analyze Relationship to Neighbors, Test Answer Choices
- Step 2's two-question framework ("How does it relate to the paragraph before it?" and "How does it set up the paragraph after it?") is practical and specific
- The "bridge" metaphor -- "define its job as a 'bridge' in the author's argument" -- is memorable and actionable
- Elimination criteria (Too Narrow, Inaccurate Function) are practical
- Tie-Breaker #1 (Blueprint Check) with the A→B→C framing is excellent -- it makes paragraph function feel like completing a logical sequence
- Tie-Breaker #2 (Focus on the Verbs) teaches precision in evaluating abstract descriptions

**Issues:**
- **No question-card.** The method is solid but entirely theoretical.
- **No worked example.** The lesson does not include even a hypothetical passage structure demonstrating the three-step method.
- **Step 1 (summarize content) could create a trap.** Students might focus too heavily on what the paragraph says rather than what it does. A brief warning would help reinforce the distinction.
- **The verb-based tie-breaker is strong but needs an example.** Showing the difference between "elaborating" and "providing an example" applied to a concrete paragraph would make it more actionable.

**Suggested edits:**
- Add a brief worked example: present a 4-paragraph passage outline and demonstrate the 3-step method on the second paragraph.
- Add a question-card after the method section.
- Add a warning after Step 1: "Remember, your content summary is a starting point. The question asks what the paragraph does, not what it says."
- Add a concrete example to the verb-based tie-breaker showing two descriptions applied to the same paragraph.

## Structural Issues

### 1. No Practice Questions

Both lessons are pure instruction. The student reads about paragraph function analysis without ever applying it.

**Recommendation:** Add at least one question-card to 36-2.

### 2. Missing Function Hierarchy Context

Modules 35, 36, and 37 form a natural analytical hierarchy (phrase → paragraph → passage), but none of them explicitly acknowledges this relationship. Students studying them in sequence would benefit from understanding how the same core skill -- identifying authorial purpose -- scales across different units of text.

**Recommendation:** Add a brief hierarchy note to 36-1 showing the progression from M35 to M36 to M37.
