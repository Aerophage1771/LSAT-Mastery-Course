# Module 38: Relationship -- Website Content Improvement Report

**Scope:** The 2 website lessons (38-1 through 38-2) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 38 covers Relationship questions -- a specialized type that almost exclusively appears in comparative passage sets. At only 18 appearances, this is a low-frequency question type, but it tests a high-value synthesis skill. The two lessons provide solid content with good comparative-passage awareness:

1. **Zero practice questions** -- neither lesson includes a question-card or worked example with actual passage material.
2. **The module correctly scopes to comparative passages** -- the stem identification breakdown and the step-by-step approach both focus on dual-passage analysis, which is the primary context for this question type.
3. **The three-step method is well-designed** -- Summarize Each Component, Define the Interaction, Match to Answer Choices is a clean, logical progression.
4. **The Directionality Test tie-breaker is excellent** -- it addresses one of the most common errors on Relationship questions: getting the direction of the relationship backwards.
5. **Low frequency but high skill transfer** -- the module correctly notes that synthesis and comparison skills transfer to other comparative passage questions.

## Lesson-by-Lesson Review

### Lesson 38-1: Germaine's Guide to RC Relationship Questions (~14 blocks)

**Role:** Module introduction, concept overview, and stem identification guide.

**Strengths:**
- Clear opening that correctly defines the task: articulating how two components interact
- The four-item interaction list (agreement/disagreement, general/specific, critique/counterpoint, expansion of scope) covers the most common relationship types
- Three stem categories (Direct Inquiry, Comparative Difference, Specific Connection) are well-differentiated
- The Specific Connection category is particularly useful -- it shows how some Relationship questions target specific claims rather than whole passages
- The Overarching Takeaway correctly emphasizes structural analysis over content summary
- Frequency note (18 appearances) and comparative-passage context set appropriate expectations

**Issues:**
- **No practice question.** The interaction types are listed but never tested.
- **No examples of the four interaction types in action.** Each type is defined abstractly but not illustrated with passage excerpts.
- **No warning about directionality errors.** The most common mistake on Relationship questions is reversing which passage does what (e.g., saying A critiques B when B critiques A). This warning is in 38-2's tie-breaker but should appear earlier.
- **The four interaction types could be presented as a diagnostic table.** A table with columns for "Passage A's Role" and "Passage B's Role" would be more scannable than a bullet list.

**Suggested edits:**
- Reformat the four interaction types as a diagnostic table with two columns.
- Add brief examples for each interaction type using hypothetical passage pairs.
- Move the directionality warning from 38-2 into 38-1 as an early callout.

---

### Lesson 38-2: Step-by-Step Approach (~16 blocks)

**Role:** The complete method for solving Relationship questions.

**Strengths:**
- Clean three-step method: Summarize Each Component, Define the Interaction, Match to Answer Choices
- Step 1's structured approach -- creating one-sentence summaries for each component -- is practical and specific
- The running example (law effectiveness: deterrence vs. economic factors) effectively demonstrates all three steps
- Step 2's six diagnostic questions (agree/disagree, general/specific, cause/effect, problem/solution, argument/critique) provide a comprehensive analytical toolkit
- The Directionality Test tie-breaker is excellent and addresses a real, frequent error
- The Point of Contact tie-breaker teaches students to find the specific intersection between components

**Issues:**
- **No question-card.** Despite the good running example, students never practice the method independently.
- **The running example is incomplete.** It demonstrates the summarization and interaction-definition steps but does not show actual answer choices being evaluated.
- **The six diagnostic questions in Step 2 are listed as a flat bullet list.** A decision-tree or flowchart structure would be more effective for guiding students through the analysis.
- **No coverage of Relationship questions within a single passage.** While most Relationship questions target comparative passages, some ask about the relationship between paragraphs within a single passage. The module does not address this variant.

**Suggested edits:**
- Add a question-card after the method section.
- Extend the running example to include answer-choice evaluation.
- Consider restructuring the Step 2 diagnostic questions as a quick decision flowchart.
- Add a brief note about within-passage Relationship questions (e.g., "relationship between the second and third paragraphs").

## Structural Issues

### 1. No Practice Questions

Both lessons are pure instruction. For a question type that tests high-value synthesis skills, the absence of practice is limiting.

**Recommendation:** Add at least one question-card to 38-2.

### 2. Comparative-Only Focus

The module focuses exclusively on comparative passage Relationship questions. While this is the primary context, some Relationship questions target paragraphs within a single passage. Adding a brief note would make the module more complete.

**Recommendation:** Add a sentence in 38-1 acknowledging within-passage relationship questions.
