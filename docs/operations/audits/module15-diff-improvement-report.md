# Module 15: Fill in the Blank -- Website Content Improvement Report

**Scope:** The 8 website lessons (15-1 through 15-8) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 15 is a well-structured unit that teaches Fill in the Blank through a clean progression: introduction, method, five pattern-specific drills, and a reference guide. The lessons are tightly scoped, consistently formatted, and use real LSAT questions with full wrong-answer analysis. The primary improvement opportunities are:

1. **No "try it first" prompts before any worked analyses.** All five drill lessons present a question card immediately followed by the analysis. A one-line pause prompt before each analysis section is the highest-priority fix across the entire module.
2. **Lesson 15-1 introduces worked content it calls "incomplete."** The bicycle infrastructure example is analyzed in a Walkthrough section, but the mastery course lesson correctly notes it will be covered in the next lesson. The website version makes this example feel self-contained, creating mild redundancy with 15-2.
3. **Lesson 15-2 drops the "Full Worked Example" from the website in the mastery version.** The website's 15-2 ends with a full worked example applying all four steps; the mastery course lesson (which is the improved version) correctly removes it and reserves that function for 15-1. The diff alternate should preserve the website's worked example for reference value.
4. **The Reference Guide (15-8) is strong but not maximal.** The mastery course reference is richer than the website version -- it adds a Support Spectrum table, analogy translation notes, and an extended reading strategy. The website version is the baseline to improve from, not the destination.
5. **Lesson 15-4 (Analogy) has a structural quirk.** The question card appears *after* the Full Analysis header rather than before it, which means the student sees the analysis framing before seeing the question. This is a layout issue in the website version.

## Lesson-by-Lesson Review

### Lesson 15-1: Introduction (35+ blocks)

**Role:** Module introduction and conceptual foundations.

**Strengths:**
- Clear Question Goal statement up front
- The Fill in the Blank vs. Main Conclusion comparison table is excellent and unique to this module -- this contrast is not made as explicitly anywhere else
- The Two Types of Blanks section (Conclusion vs. Premise) with indicator lists is actionable and well-organized
- Common Conclusion Patterns breakdown (Causal, Conditional, Comparative, Normative) provides a useful taxonomy before students encounter the drills
- The Core Principle callout ("Read Everything Before the Blank as Premises") is the most important strategic insight in the module and is placed prominently
- The Worked Example (bicycle infrastructure) shows a complete application before any drill

**Issues:**
- **No "try it first" prompt.** The Worked Example walks through a stimulus without asking the student to attempt classification or prediction before reading the Walkthrough. Even a brief "Before reading the walkthrough, classify the blank and form your own prediction" would improve engagement.
- **Question Goal is slightly misleading.** The website states "The blank almost always represents the argument's conclusion" -- which is accurate but does not prepare students for the Premise Blank variant covered two paragraphs later. The mastery course version improves this with a more balanced framing.
- **Common Conclusion Patterns section is almost identical to the Step 2 "Identify the Reasoning Pattern" section in 15-2.** Both lessons present Causal Chain, Analogy/Rule Application, Problem-Solution, and Paradox Resolution patterns with near-identical language. This duplication means students encounter the same content twice in back-to-back lessons.
- **The "Frequency" callout is placed before the "Recognizing the Question" section** in the website version. In the mastery course version it is placed after Recognizing the Question, which is more logical (you recognize it, then you learn how often it appears). The mastery course ordering is better.

**Suggested edits:**
- Add a "Try this before the walkthrough" prompt immediately before the Walkthrough section.
- Trim the Common Conclusion Patterns section to a two-sentence forward reference ("You will see these patterns in depth in the drills") since 15-2 covers them in full.
- Reorder so that the Frequency callout follows Recognizing the Question rather than preceding it.

### Lesson 15-2: The 4-Step Method (40+ blocks)

**Role:** Complete, repeatable method for Fill in the Blank.

**Strengths:**
- Excellent structure: overview process block, then each step gets its own H2 with explanation, bullet list, and callout
- The Timing callout (1 minute 20 seconds) is concrete and action-oriented
- The Litmus Test callout (insert your answer and re-read) is a genuinely useful verification habit
- The Trap Types breakdown (Too Strong, Out of Scope, Reversed Logic, Mere Restatement, Wrong Relationship) is a strong taxonomy with memorable labels
- "When Two Answers Look Good" callout addresses the most common real test-day scenario

**Issues:**
- **No "try it first" prompt.** The Full Worked Example (psychologist/effort praise) is presented as a Walkthrough without asking students to attempt the question first. The question is embedded in the blockquote but no prompt separates it from the analysis.
- **The Reasoning Patterns breakdown in Step 2 duplicates 15-1.** As noted above, both lessons list the same five patterns. The mastery course version of 15-2 correctly does not include the worked example (reserving that for 15-1), but both still include the patterns list.
- **"Full Worked Example" heading creates a false landmark.** The example does not include a question-card with answer choices -- it's a blockquote + walkthrough. A student who wants to attempt the question before seeing the analysis has no answer choices to work with. The worked example would be stronger as a question-card.
- **The "Common Mistakes" section at the end of the website version** (Reading answers before prephrasing, Real-world answers, Ignoring hedging, Failing to re-read) is missing from the mastery course version's content array but is present in the mastery course's diff alternate. It should be in the main content because it summarizes the most common errors concisely.

**Suggested edits:**
- Add a "Try this before reading the walkthrough" prompt before the Walkthrough.
- Convert the Worked Example from a blockquote to a question-card so students have answer choices to evaluate.
- Add the Common Mistakes breakdown to the main content (not just the diff alternate).
- Consider trimming the Reasoning Patterns breakdown in Step 2 to a cross-reference ("See the pattern descriptions in Lesson 1") to reduce duplication with 15-1.

### Lesson 15-3: Drill -- Mechanistic Chain (35+ blocks)

**Role:** Pattern-specific drill with a real LSAT question (PT-102-S-2-Q-3).

**Strengths:**
- The Intention-Mechanism Split callout is the single best insight in this lesson -- it names the central tension that this pattern type exploits
- The "Trace the Dominoes" three-phase strategy (Map → Identify Endpoint → Check for Goal Contradiction) is concrete and memorable
- The Step 1 breakdown with numbered causal steps (Triggering Action → Step 1 → Step 2 → Step 3 → Endpoint) is the clearest causal chain visualization in the module
- All five wrong answers are fully analyzed with labeled trap types (Out-of-Scope Comparison, Contradicts Premise, Double Contradiction)
- The Common Traps breakdown at the end (Goal Alignment, Scope Expansion, Premise Contradiction, Direction Reversal) provides reusable pattern recognition

**Issues:**
- **No "try it first" prompt.** The question card (PT-102-S-2-Q-3) is presented and then immediately followed by the Full Analysis section. A student who wants to attempt before reading cannot do so without a deliberate pause -- the layout provides no natural stopping point.
- **Instructions say "Commit to an answer before scrolling down."** This is a good intent, but it relies on the student's self-discipline and only works on long-scroll layouts. A formal prompt block before the analysis would be more reliable.
- **The "Trace the Dominoes" strategy is labeled differently across lessons.** Lesson 15-3 calls it "Trace the Dominoes," Lesson 15-4 calls it "Build the Map," Lesson 15-5 calls it "Problem → Solution → Obstacle → Reversal," etc. These are correctly different for different patterns -- this is not a flaw -- but the Reference Guide does not surface these strategy names, which it should.

**Suggested edits:**
- Add a diff-note callout block before the Full Analysis section: "Attempt the question above before reading the analysis."
- Add the per-pattern strategy names (Trace the Dominoes, Build the Map, etc.) to the Reference Guide's Pattern Recognition table.

### Lesson 15-4: Drill -- Analogy Completion (40+ blocks)

**Role:** Pattern-specific drill with a real LSAT question (PT-123-S-2-Q-3).

**Strengths:**
- The Parallel Transfer Rule callout is strong -- "the correct answer is the one that preserves every mapping" is exactly the right framing
- The Build the Map three-phase strategy is well-structured and the component table (Source Actor, Source Action, Source Object, Target Actor, Target Action) makes the mapping explicit
- The Mapping Breakdown table is the most analytical wrong-answer analysis in the module -- it tests each element independently (Action, Object, Direction, Scope) with a binary check
- Answer (E)'s failure on scope-narrowing is a sophisticated distinction that many students miss; the analysis handles it well

**Issues:**
- **No "try it first" prompt.** The question card appears after "Full Analysis" framing, meaning the heading "Full Analysis" is visible before the student has seen the question. The layout is reversed: the heading appears, then the question card, then the analysis. The question card should precede any analysis framing.
- **The "trap gradient" observation** ("(A) and (E) are the most tempting because they each preserve most of the mapping") is the most useful meta-observation in the lesson, but it is buried in a paragraph after the table. It should be promoted to a callout or named insight.

**Suggested edits:**
- Restructure the layout: Problem → Question Card → [Pause Prompt] → Full Analysis. The question card currently appears inside the Full Analysis section.
- Promote the "trap gradient" observation to a named callout.

### Lesson 15-5: Drill -- Problem and Solution (40+ blocks)

**Role:** Pattern-specific drill with a real LSAT question (PT-125-S-2-Q-7).

**Strengths:**
- The Obstacle Reversal Rule callout is precise: "The correct answer always mirrors the obstacle -- it does not introduce new considerations."
- The Problem → Solution → Obstacle → Reversal four-part framework is the clearest structural decomposition in the drill lessons
- The word "singlehandedly" is correctly identified as the fulcrum of the argument -- this level of close reading guidance is genuinely useful
- Answer (E)'s "overkill" label is a named trap type not seen in other lessons; it correctly identifies answers that solve the problem but go beyond what the argument requires

**Issues:**
- **No "try it first" prompt.** Same pattern as 15-3 and 15-4 -- the analysis follows immediately.
- **The question card has a formatting inconsistency.** It is preceded by a `<br/>` paragraph block (a visual spacer) before the analysis header, which is present in 15-4 and 15-6 but not 15-3. This suggests the spacers were added manually to create scroll distance on long-scroll layouts. A formal pause prompt would replace this function properly.
- **The stimulus uses bolded text ("Environmentalist:")** as an attribution label. This is consistent with the real LSAT format, but the analysis never notes that the attribution label is part of the question type pattern (some FitB questions have named speakers; some don't). Worth a brief note.

**Suggested edits:**
- Add a "Try this before the analysis" prompt after the question card.
- Remove the `<br/>` spacer paragraphs and replace with a genuine pause prompt.

### Lesson 15-6: Drill -- Philosophical Rebuttal (40+ blocks)

**Role:** Pattern-specific drill with a real LSAT question (PT-114-S-1-Q-19).

**Strengths:**
- The Redefinition Rule callout is the clearest articulation of the pattern's core move: "the author takes the opponent's key term and redefines it in terms of the author's own core concept"
- The Thesis → Challenge → Rebuttal → Consequence four-part decomposition parallels the Problem → Solution → Obstacle → Reversal framework in 15-5, creating a nice structural consistency
- The "reductive strategy" insight ("the essayist's strategy is reductive: they take the opponent's concept and show it can be fully explained in terms of the essayist's own concept") is an advanced logical observation handled at the right level of detail
- Answer (B)'s failure as a thesis contradiction is correctly identified -- this is a non-obvious trap that many students would select

**Issues:**
- **No "try it first" prompt.** Same pattern.
- **The "Abstraction Mismatch" trap** (answer shifts from philosophical analysis to practical advice) is named but the lesson doesn't explain *why* this matters structurally. The explanation would benefit from one sentence: "Philosophical arguments about the nature of value conclude with claims about the nature of value -- not with advice about how to obtain it."
- **The lesson title says "Philosophical Rebuttal" but the analysis uses "dialectical argument."** Both terms are correct, but the inconsistency could confuse students who try to look up the strategy by name. The mastery course version should standardize on one term.

**Suggested edits:**
- Add "try it first" prompt.
- Add one sentence explaining why the level-of-abstraction check matters.
- Standardize on "Philosophical Rebuttal" as the canonical pattern name.

### Lesson 15-7: Drill -- Conditional Logic and Paradox (45+ blocks)

**Role:** Pattern-specific drill with a real LSAT question (PT-123-S-3-Q-16).

**Strengths:**
- The Paradox Articulation Rule callout is the strongest strategic insight in the lesson: "many wrong answers attempt to escape the paradox... the correct answer does none of these"
- The Fact → Condition → Conflict → Paradox four-part framework correctly identifies the paradox's logical structure and leads directly to the pre-phrase
- The "escape-route closer" observation ("nothing else could prompt people to make the sacrifices") is a sophisticated structural observation that directly explains why Answer (A) fails
- "Obviously, then" as a conclusion signal is correctly identified and explained
- The "descriptive-to-normative shift" trap (the answer says "should" when the argument is drawing a factual conclusion) is an important cross-question-type pattern

**Issues:**
- **No "try it first" prompt.** Same pattern.
- **The lesson covers both "Conditional Logic" and "Paradox" in the title, but the question is a paradox question, not a conditional logic question.** The conditional logic framing (if-then chains, triggering facts) is discussed in the strategy section but does not appear in the drill. A second example showing a pure conditional chain would make the "Conditional Logic" part of the title accurate.
- **Answer (C)'s failure ("follows from Premise 1 only") is the most important pedagogical point in the lesson.** It shows that an incomplete synthesis -- one that ignores half the premises -- produces a wrong answer even if that answer is logically true. This insight should be elevated to a callout.

**Suggested edits:**
- Add "try it first" prompt before Full Analysis.
- Elevate the "partial conclusion" insight to a named callout.
- Consider adding a short second example showing a pure conditional chain to justify the lesson title.

### Lesson 15-8: Reference Guide (60+ blocks)

**Role:** One-stop quick-reference for the entire module.

**Strengths:**
- The mastery course version significantly outperforms the website baseline: it adds the Support Spectrum Quick-Check table, an Analogy and Comparative Translation Reminder, and an extended reading strategy with a "clause-first" step for buried indicator words
- The 7-item Trap taxonomy is comprehensive and includes "Too Weak" (absent from 15-2's version) and "Addresses Wrong Part of Argument" -- both important traps not flagged elsewhere
- The 10-step Test Day Checklist is the most complete process guide in the module and includes the insert-and-re-read verification step
- The Pattern Recognition Cheat Sheet with pre-phrase templates for all six patterns is the best single tool in the reference guide

**Issues (relative to website baseline):**
- **The website version's Section 6 (Reading Strategy) is a 4-step process.** The mastery course version improves this to a 5-step process by adding the "interrupted indicator" step (when "since" or "therefore" is buried inside a clause). This improvement should be preserved in the diff alternate.
- **The website version does not include a Support Spectrum table.** The mastery course adds this (Supported / Merely Consistent / Anti-Supported). This is the most important addition in the mastery course reference -- it gives students a vocabulary for ranking answer choices that the website version lacks entirely.
- **Per-pattern strategy names are not listed** in either version. The Reference Guide should surface "Trace the Dominoes" (Mechanistic Chain), "Build the Map" (Analogy), "Problem-Solution-Obstacle-Reversal" (Problem-Solution), "Thesis-Challenge-Rebuttal-Consequence" (Philosophical Rebuttal), and "Fact-Condition-Conflict-Paradox" (Conditional/Paradox) as named strategies with cross-references to their drill lessons.
- **No frequency data.** The website 15-1 mentions that FitB appears "roughly 1-2 times per scored section." This should appear in the reference guide for pre-test review.

**Suggested edits:**
- Add the Support Spectrum table (it exists in the mastery course content but not in the website baseline).
- Add a per-pattern named strategy column to the Pattern Recognition table.
- Add a one-line frequency note near the top of the reference guide.

## Structural Issues

### 1. "Try It First" Gap Across All Five Drill Lessons

The module's five drill lessons (15-3 through 15-7) all present a question card followed immediately by the analysis. No lesson includes a formal prompt asking the student to attempt the question before reading the walkthrough. The website lessons note "Commit to an answer before scrolling down" in the Instructions line, but this is easily missed and depends on layout.

**Recommendation:** Add a `{{+**Try this:** Attempt the question before reading the analysis below.+}}` block before every analysis section in every drill lesson. This is the highest-priority structural change in the module.

### 2. Pattern Description Duplication Between 15-1 and 15-2

Both 15-1 (Common Conclusion Patterns) and 15-2 (Step 2: Identify the Reasoning Pattern) present the same five argument patterns with nearly identical descriptions. A student who reads both lessons in sequence encounters the same content twice with no new depth the second time.

**Recommendation:** In 15-1, trim Common Conclusion Patterns to a 2-sentence forward reference. Let 15-2 own the full pattern descriptions. The mastery course version of 15-1 already improves this slightly, but the diff alternate should make the reduction explicit.

### 3. Layout Inversion in Lesson 15-4

In 15-4, the question card appears *inside* the "Full Analysis" section rather than before it. This means the section heading "Full Analysis" is visible before the student has seen the question, which primes them to read rather than attempt. This is a unique structural flaw in 15-4 not present in other drill lessons.

**Recommendation:** The diff alternate for 15-4 should note the inversion and suggest moving the question card above the analysis framing.
