# Module 18: Must Be False -- Website Content Improvement Report

**Scope:** The 8 website lessons (18-1 through 18-8) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 18 is a well-structured and pedagogically strong unit covering one of the rarest but most precise LSAT question types. The 8 lessons follow a clear arc from introduction through four drill patterns, an advanced lesson, and a comprehensive reference guide. The drills (Lessons 18-3, 18-4, 18-5) each pair a real PT question with a thorough worked analysis. The primary improvement opportunities are:

1. **No "try it first" prompt before any worked analysis** -- Lessons 18-3, 18-4, 18-5, and 18-7 all transition directly from the question-card into the full analysis with no pause for independent attempt. This is the module's most impactful missing feature.
2. **Lesson 18-6 (Numerical Impossibility) is missing its question-card** -- the stimulus text is absent. The lesson provides a "Practice Problem" header followed by blank space, then jumps into analysis referencing a 200-employee survey problem that was never presented. The student cannot attempt the question.
3. **Lesson 18-7 (Advanced) has no real PT question** -- the complex contradiction drill uses a constructed (non-PT) example about Thornfield Academy. While pedagogically sound, it is the only drill in the module without a real LSAT question-card.
4. **The Reference Guide (18-8) is excellent** -- at ~70 blocks it is the most thorough reference guide in the course, covering the 4-step method, all 5 contradiction types, MBF vs. MBT comparison, EXCEPT strategy, quantifier reference, 7 common traps, and a 10-step test-day checklist.
5. **Lesson 18-2 (Step-by-Step) has no practice question** -- the method lesson describes a worked snapshot using a constructed example but never offers a question-card for the student to apply the method.

## Lesson-by-Lesson Review

### Lesson 18-1: Introduction (14 blocks)

**Role:** Module introduction and overview.

**Strengths:**
- Clear Question Goal statement opens the lesson with the exact task: find the logically impossible answer.
- The "Why This Matters" section effectively motivates the topic by connecting MBF to broader contradiction-detection skills.
- The Question Stem table (5 rows) is comprehensive, covering CANNOT, must be false, EXCEPT, incompatible, and violates phrasings.
- The MBF vs. MBT comparison list (3 items) immediately frames the question type relative to its mirror image, preventing confusion.
- The Impossibility Standard paragraph is the single most important conceptual statement in the module -- the "zero room for coexistence" framing is precise.
- The Concept Snapshot (shelter/Rex example) is simple, relatable, and demonstrates rule-chaining in 3 sentences.
- The EXCEPT Format section addresses the most common phrasing students encounter, with practical guidance on flipping the instinct.
- The Common Reasoning Patterns list (4 items) previews all four drill types cleanly.

**Issues:**
- **No practice question or mini-exercise.** The lesson is 14 blocks of pure theory. Even a single "identify the impossible statement" micro-exercise after the Concept Snapshot would improve active learning.
- **The Common Reasoning Patterns list previews topics covered in Lessons 18-3 through 18-6 but does not mention the Temporal Impossibility pattern** that the Reference Guide later lists as a separate type. This creates a minor inconsistency.

**Suggested edits:**
- Add a 2-3 sentence micro-exercise after the Concept Snapshot asking the student to identify the impossible statement from a new mini-stimulus.
- Add "Temporal Impossibilities" to the Common Reasoning Patterns list, or note that temporal patterns are a subtype of Exclusive Boundaries.

### Lesson 18-2: Step-by-Step Guide (12 blocks)

**Role:** The complete 4-step method for solving Must Be False questions.

**Strengths:**
- The 4-Step Method process block is clear and actionable: Diagram Rules, Test Each Answer, Find the Contradiction, Verify Impossibility.
- The "Could Be True" Elimination Technique (Steps A-B-C) gives a concrete, repeatable sub-process for the testing phase.
- The Worked Method Snapshot (certified instructor / beginner sessions) is a compact, effective demonstration of rule-chaining.
- The Wrong-Answer Patterns list (4 items) is one of the strongest teaching tools in the lesson -- each trap is named, described, and distinguished from the correct answer type.
- The Timing Tip callout gives a concrete time-allocation strategy (60% on rule-diagramming).

**Issues:**
- **No question-card.** The method lesson describes the process but never lets the student apply it. The Worked Method Snapshot is a prose walkthrough, not an interactive exercise.
- **The method steps in 18-2 differ slightly from those in the Reference Guide (18-8).** The 18-2 version uses "Diagram the Rules / Test Each Answer / Find the Contradiction / Verify Impossibility." The 18-8 version uses "Check the Facts and Find the Rules / Combine Rules and Prephrase / Scan Answers for a Conflict / Eliminate Traps." These are the same underlying process but with different naming. Students may be confused by the inconsistency.
- **The "Could Be True" Elimination Technique overlaps with Step 2 of the 4-Step Method.** Both describe the same "can I build a compatible scenario?" test. This is mild redundancy but not harmful.

**Suggested edits:**
- Add a question-card after the Worked Method Snapshot so the student can apply the 4-step method before moving to the drill lessons.
- Align the step names between 18-2 and 18-8 so the student learns one consistent vocabulary. The 18-8 names are more precise ("Combine Rules and Prephrase" is more instructive than "Test Each Answer").
- Consider renaming the 18-2 steps to match 18-8 or adding a note explaining the equivalence.

### Lesson 18-3: Drill -- Conditional Breakers (35 blocks)

**Role:** First drill lesson. Teaches conditional violation pattern with PT-103-S-1-Q-13.

**Strengths:**
- Strong conceptual opening: the "exactly one way to break a conditional" framing is precise and memorable.
- The ordered list (identify sufficient, identify necessary, the contradiction is P and not-Q) gives a mechanical procedure.
- The "Translate Absolutes into Conditionals" strategy callout is immediately actionable.
- The question-card (PT-103-S-1-Q-13) is well-chosen: the politician/honesty stimulus buries the key rule in the final sentence, testing extraction skill.
- The Step-by-Step analysis (Extract Rules, Diagram, Evaluate Each Answer) mirrors the 4-step method.
- The breakdown component for each stimulus statement (Background, Conclusion, Premise, Key Rule badges) is visually clear and teaches students to categorize before solving.
- Full 5-answer evaluation with Possible/Correct badges and substantive explanations.

**Issues:**
- **No "try it first" prompt before the analysis.** The lesson includes "Read the stimulus and commit to an answer before scrolling down" as an instruction, which is good. However, there is no visual separator or callout between the question-card and the analysis that reinforces this pause. The analysis begins immediately after the `<br/>` spacers and hr, which is easy to scroll past.
- **The converse/inverse paragraph** ("Common distractors present the converse...") repeats the sufficient/necessary distinction that the ordered list just established. Minor redundancy.

**Suggested edits:**
- Add a prominent callout or styled block before the Full Analysis section: "**Try this:** Attempt the question before reading the analysis below."
- Consolidate the converse/inverse paragraph into the ordered list as a "watch out for" note.

### Lesson 18-4: Drill -- Quantifier Clashes (31 blocks)

**Role:** Second drill lesson. Teaches quantifier contradiction pattern with PT-113-S-3-Q-8.

**Strengths:**
- The Quantifier Hierarchy (3-item ordered list) is pedagogically excellent -- it distinguishes contradictory, sub-contrary, and entailment relationships clearly.
- The "Find the Universal, Then Negate It" strategy callout gives a purely mechanical approach that works on the most common MBF pattern.
- The question-card (PT-113-S-3-Q-8) is well-chosen: the biological stimulus (eggs/reproduction) has a universal rule in the very first sentence, testing whether the student can extract it cleanly.
- The breakdown analysis for each stimulus statement (Universal Rule, Qualifier, Example, Context badges) teaches students to separate load-bearing rules from context.
- The evaluation of answer (C) is notable for catching a subtle trap: "rare" describes the specific salamander, not all female-only species.

**Issues:**
- **No "try it first" prompt.** Same issue as 18-3. The instruction paragraph says to "commit to an answer before scrolling down," but there is no visual callout reinforcing this.
- **The question-card placement is odd.** In this lesson, the `<br/><br/><br/>` spacers and hr come *before* the question-card, creating a gap between the "The Problem" header and the actual question. In 18-3, the spacers come *after* the question-card. This inconsistency is confusing.

**Suggested edits:**
- Add a "Try this" callout before the Full Analysis section.
- Move the `<br/>` spacers to after the question-card (consistent with 18-3) so the spacing separates the question from the answer, not the header from the question.

### Lesson 18-5: Drill -- Exclusive Boundaries (36 blocks)

**Role:** Third drill lesson. Teaches multi-constraint combination with PT-102-S-4-Q-15.

**Strengths:**
- The opening paragraph clearly distinguishes Exclusive Boundaries from the two prior patterns (Conditional Breakers and Quantifier Clashes), building on the progression.
- The 3-step "Building the Logical Map" process (Extract, Layer, Identify the forbidden zone) is clean and actionable.
- The "Combine, Don't Isolate" strategy callout captures the essential insight: no single sentence produces the contradiction.
- The question-card (PT-102-S-4-Q-15) is the strongest in the module: the dinosaur/plesiosaur stimulus requires combining geographic, temporal, and biological constraints, testing true synthesis skill.
- The constraint-extraction breakdown (Geographic, Biological, Temporal, Plesiosaur Facts) with distinct badges is the most thorough stimulus decomposition in the module.
- The "Forbidden Zone" callout (Dinosaurs + Southern Half + Cretaceous Period = Impossible) is the single best teaching moment in the module -- it gives the student a compact formula they can visualize.
- The evaluation of answer (C) notes it must be *true*, not just possible -- a sophisticated distinction that reinforces MBF logic.

**Issues:**
- **No "try it first" prompt.** Same issue as 18-3 and 18-4.
- **Same question-card placement inconsistency as 18-4** -- spacers before the question-card rather than after.

**Suggested edits:**
- Add a "Try this" callout before the Full Analysis.
- Fix the spacer placement for consistency.

### Lesson 18-6: Drill -- Numerical Impossibility (29 blocks)

**Role:** Fourth drill lesson. Teaches overlapping-group mathematics.

**Strengths:**
- The Core Principle section (100-person room example) is the clearest explanation of forced overlap in the course. The general formula (X + Y - 100% = minimum overlap) is immediately usable.
- The "Spotting Numerical Impossibility" strategy callout is actionable and specific.
- The Types of Numerical Constraints breakdown (Overlapping Majorities, Fixed Totals, Exhaustive Partitions, Rate-Based Constraints) with frequency badges covers every pattern the student might encounter.
- The analysis of the 200-employee problem is thorough: deriving the forced overlap (70 employees), then testing each answer against that constraint.
- The evaluation of answer (E) is notably honest -- it acknowledges that (E) is also technically impossible (overlap of 60 < minimum of 70) but explains why (D) is the intended correct answer.

**Issues:**
- **The question-card is missing.** The lesson has a "Practice Problem" header followed by `<br/>` spacers and an hr, then jumps directly to "Full Analysis" -- but the actual stimulus text, question, and answer choices are never presented. The analysis references "200 employees," "75% commute by car," "60% work in headquarters," but the student has no question to attempt. This is the module's most critical content gap.
- **No "try it first" prompt.** Even if the question-card were present, there is no pause callout.
- **The lesson starts with an H3 ("Drill: Numerical Impossibility") rather than an H2.** This is inconsistent with all other drill lessons, which open with an H2.

**Suggested edits:**
- **High priority:** Add the missing question-card with the full stimulus, question stem, and 5 answer choices for the 200-employee commuter problem.
- Add a "Try this" callout before the Full Analysis.
- Change the opening heading to H2 for consistency.

### Lesson 18-7: Advanced -- Complex Contradictions (42 blocks)

**Role:** Advanced lesson covering multi-rule, distractor-laden, contrapositive-dependent contradictions.

**Strengths:**
- The Three Difficulty Factors framework (Multi-Step Deductions, Embedded Distractors, Conditional Reversals) is excellent -- it gives students a diagnostic vocabulary for understanding *why* hard questions are hard.
- Each factor gets a concrete example in a callout box (electrician chain, museum scheduling, Dr. Hale's paper). These are simple, self-contained, and illustrate the factor clearly.
- The Complex Contradiction Drill (Thornfield Academy merit scholarships) is the most sophisticated analysis in the module. The 6-step deductive chain process block is a model of how to derive a hidden constraint.
- The "Why This Is Hard" callout explicitly maps the drill to all three difficulty factors, reinforcing the framework.
- The four advanced strategies (Build Full Chain, Flag Distractors, Test Contrapositives, Two-World Test) are practical and specific.
- The "Chain-and-Block Pattern" tip callout names the single most common hard-MBF structure and gives a one-sentence recognition heuristic.
- The wrong-answer patterns table (5 rows) is comprehensive and parallel to the Reference Guide.

**Issues:**
- **No real PT question-card.** The Thornfield Academy problem is a constructed example, not a real LSAT question. This is the only drill in the module that does not reference an actual PrepTest question. While the example is pedagogically sound, a real PT question would be more authoritative.
- **No "try it first" prompt.** The lesson says "Commit to an answer before reading the analysis" in a paragraph, but there is no visual callout.
- **The lesson is very long (42 blocks).** The combination of three difficulty factors, three concrete examples, a full drill with analysis, four strategies, and a wrong-answer table makes this a dense read. Consider whether the strategies section could be moved to the Reference Guide to reduce length.
- **The analysis of answer (C)** is notably long and hedging ("However, the most natural reading creates a tension..."). While accurate, this level of uncertainty in a teaching context may confuse students. A shorter, more decisive explanation would be clearer.

**Suggested edits:**
- Add a real PT question-card for the Complex Contradiction Drill if a suitable question is available.
- Add a "Try this" callout before the analysis.
- Consider moving the four advanced strategies to the Reference Guide (18-8) and replacing them with a shorter summary in 18-7.
- Tighten the analysis of answer (C) to be more decisive.

### Lesson 18-8: Reference Guide (~70 blocks)

**Role:** Complete reference for the entire module.

**Strengths:**
- **The most comprehensive reference guide in the course.** At ~70 blocks, it covers:
  - The 4-Step Method (process block + timing callout)
  - MBF Contradiction Types table (5 types with lesson cross-references)
  - MBF vs. MBT Comparison table (8 features, side-by-side)
  - The EXCEPT Strategy (5-step process + shortcut callout)
  - Quantifier Contradiction Reference table (4 contradiction pairs)
  - The Quantifier Square callout (contradictory, contrary, subcontrary pairs)
  - Common Traps breakdown (7 traps with badges and detection tests)
  - Test Day Checklist (10-step process block)
  - Two Quick-Reference Tables (Contradiction Type Quick-Check, Impossibility Spectrum)
  - Module Summary callout
- The MBF vs. MBT comparison table is the best side-by-side reference in the course -- 8 features compared with precise, parallel language.
- The Impossibility Spectrum table (0% through 100%) is a brilliant teaching tool -- it explicitly maps the certainty continuum and marks which levels are/are not MBF answers.
- The 7 Common Traps section is more detailed than what appears in any individual lesson, making the reference guide genuinely additive.
- Cross-references to specific lessons in the Contradiction Types table help students navigate back for review.

**Issues:**
- **The Numerical Impossibility contradiction type references "Lessons 3-5" in the cross-reference column, but the dedicated numerical lesson is Lesson 6.** This should reference Lesson 6 (or "Lesson 6" primarily with 3-5 as supplementary).
- **The 4-Step Method step names differ from those in Lesson 18-2** (see 18-2 review above). The 18-8 names are better, but the inconsistency could confuse students who studied 18-2 first.
- **No practice question.** A reference guide does not typically need one, but a single "quick-check" exercise at the end would help students verify they can apply the reference under time pressure.
- **The Quantifier Square callout could include a visual diagram** (even a text-based one) showing the square of opposition. The prose description is accurate but a diagram would be faster to scan.

**Suggested edits:**
- Fix the Numerical Impossibility lesson cross-reference to include "Lesson 6."
- Align the 4-Step Method naming with Lesson 18-2 (or vice versa).
- Consider adding a mini-exercise at the very end.
- Consider adding a text-based square of opposition diagram.

## Structural Issues

### 1. Missing "Try It First" Prompts (HIGH PRIORITY)

Every drill lesson (18-3, 18-4, 18-5, 18-6, 18-7) provides a worked analysis that begins immediately after the question presentation. None includes a prominent visual callout encouraging the student to attempt the question independently before reading the solution. The text instructions ("commit to an answer before scrolling down") are present in 18-3, 18-4, and 18-5, but they are embedded in a regular paragraph and easy to miss.

**Recommendation:** Add a styled callout block (e.g., `variant: "tip"` or `variant: "default"`) before every "Full Analysis" section that reads: "**Try this:** Attempt the question before reading the analysis below." This is the single highest-impact change for the module.

### 2. Missing Question-Card in Lesson 18-6

The Numerical Impossibility drill has no question-card. The student sees a "Practice Problem" header, blank space, and then an analysis referencing data they have never seen. This makes the lesson unusable for independent practice.

**Recommendation:** Add the full question-card (stimulus, question stem, 5 answer choices) for the 200-employee commuter problem. This is the second highest-priority fix.

### 3. Method Step Naming Inconsistency

The 4-step method is described with different step names in 18-2 and 18-8. This creates unnecessary friction for students who study both lessons.

**Recommendation:** Use the 18-8 step names throughout (Check Facts/Find Rules, Combine/Prephrase, Scan for Conflict, Eliminate Traps) as they are more descriptive and action-oriented.

### 4. Question-Card Spacing Inconsistency

Lessons 18-3, 18-4, 18-5, and 18-6 use different patterns for the `<br/>` spacer placement relative to the question-card. In 18-3, spacers follow the card; in 18-4, 18-5, and 18-6, spacers precede the card (or the card is absent). Standardize to: question-card first, then spacers, then analysis.
