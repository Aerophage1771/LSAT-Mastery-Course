# Module 5: Flaw -- Website Content Improvement Report

**Scope:** The 12 website lessons (5-1 through 5-12) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 5 is a comprehensive, well-structured unit that covers the single most important LR question type. The 12 lessons progress logically from introduction through advanced concepts, and lessons 5-4 through 5-11 include worked examples with full answer-choice breakdowns. The primary improvement opportunities are:

1. **Lesson 5-1 is overloaded** -- it covers introduction, the Flaw Mindset, the Gap Model, a full flaw category preview table, indicator words, AND a walkthrough example. At ~45 blocks this is too much for a first encounter.
2. **Lesson 5-3 (Field Guide) heavily overlaps 5-1** -- both contain flaw category tables with frequency data. The Field Guide has a 19-row Master Taxonomy that replicates and expands the 10-row preview in 5-1.
3. **No question-cards in any lesson** -- despite extensive worked examples and practice snippets, no lesson uses an actual question-card component. All practice is in blockquotes with accordion reveals or inline process blocks.
4. **Drill lessons (5-4 through 5-7) are the strongest section** -- each isolates one flaw family, provides a full worked example with answer-choice breakdown, and includes a second practice question. This is the gold standard for the module.
5. **The Reference Guide (5-12) is excellent and comprehensive** -- at roughly 100+ blocks with full taxonomy, LSAT decoder, traps, flowchart, conditional reference, and evaluation strategies, it is the strongest reference guide in the course. No structural issues.

## Lesson-by-Lesson Review

### Lesson 5-1: MODULE 5: Flaw (~45 blocks)

**Role:** Module introduction, the Flaw Mindset, and the Gap Model.

**Strengths:**
- Strong opening with precise Question Goal statement and frequency/timing data
- The Flaw vs. Method of Reasoning comparison table is clear and actionable
- The Gap Model is the single best conceptual framework in the lesson -- the pillar/foundation/roof metaphor is memorable
- The "Accept the Premises" / "Fact vs. Interpretation" breakdown is critical pedagogy that many prep courses skip
- The breakfast/weight-loss walkthrough of the Gap Model is concrete and effective
- The flaw category preview table with approximate frequencies and priority ratings gives students a roadmap
- Indicator Words table is useful for pattern-matching on exam day

**Issues:**
- **Scope creep:** The lesson tries to be introduction, conceptual framework, flaw category preview, indicator reference, AND first worked example all in one. That is five distinct pedagogical goals.
- **The flaw category preview table (10 rows)** is redundant with the Master Taxonomy in Lesson 5-3 (19 rows). Students see two overlapping tables before encountering any practice.
- **"Your First Flaw Question" uses a constructed example**, not a real LSAT question. The ad hominem politician example is useful for illustration, but there is no question-card with answer choices the student can attempt.
- **The "Flaw Mindset" section** (blocks on Description vs. Evaluation) is well written but could be shorter. The MoR-to-Flaw transition is a single conceptual point stretched across multiple blocks.

**Suggested edits:**
- Move the flaw category preview table and indicator words table to Lesson 5-3 (or cut them entirely, since 5-3 covers everything in more depth). Replace with a brief "You'll learn 10+ flaw categories in the Field Guide" forward reference.
- Add a question-card after the politician walkthrough so students can attempt an actual question.
- Tighten the "Flaw Mindset" section to one comparison table and one paragraph, not a table + breakdown + additional comparison table.

### Lesson 5-2: The 4-Step Method (~60 blocks)

**Role:** The complete solving method for Flaw questions.

**Strengths:**
- Clean 4-step structure: Identify, Gap, Name, Match
- Timing breakdown (30s / 20s / 40-60s) is specific and useful
- The Gap Analysis Technique table (7 questions) is one of the best diagnostic tools in the module
- The "Before and After" breakdown showing passive vs. critical reading is an excellent teaching device
- Prephrasing Tips with "Too Specific / Just Right / Too Abstract" calibration is actionable
- The Wrong Answer Patterns table (5 types) is comprehensive and accurate
- The full food-truck editorial walkthrough demonstrates compound flaws clearly
- The three Prephrasing Practice snippets with accordion reveals are excellent active-learning opportunities

**Issues:**
- **No question-card.** Despite being the method lesson with a full worked example and three practice snippets, there is no interactive question-card the student can attempt before seeing the answer.
- **The food-truck editorial worked example** is constructed, not from a real LSAT. Using a real PT question would add credibility.
- **The Mental Flaw Checklist (9 items)** is presented as a process block, but it overlaps heavily with the Quick Identification Flowchart in both 5-3 and 5-12. Three versions of the same checklist across the module is redundant.
- **Length.** At ~60 blocks, this is among the longest lessons. The method itself is four steps, but each step has sub-sections, examples, and callouts that extend the reading significantly.

**Suggested edits:**
- Add a question-card after the prephrasing practice section.
- Consolidate the Mental Flaw Checklist -- either keep it here and remove it from 5-3/5-12, or keep it only in 5-12 and reference it from here.
- Consider cutting one of the three prephrasing snippets to reduce length.

### Lesson 5-3: Field Guide to Common Error Types (~120 blocks)

**Role:** Comprehensive taxonomy of all flaw types with examples.

**Strengths:**
- The Master Taxonomy table (19 rows with category, type, error, and frequency) is the definitive reference for the module
- Each flaw type gets a concrete blockquote example, an explanation of why it is flawed, and exact LSAT answer-choice language
- The breadth of coverage is excellent: 5 categories, 19 sub-types, with nothing important omitted
- The LSAT Language Translation Table (16 rows mapping common names to LSAT phrasing) is one of the highest-value reference tools in the course
- The Quick Identification Flowchart (9-step process block) provides a systematic approach
- Causal Flaw Recognition Shortcut callout is practical and actionable
- Necessary vs. Sufficient Quick Reference callout is correctly placed within the conditional section

**Issues:**
- **Extreme length.** At approximately 120 blocks, this is by far the longest lesson in the module and possibly the entire course. A student reading linearly will fatigue before reaching the later flaw types.
- **No practice questions.** Despite covering 19 flaw types with examples, there is not a single practice exercise or question-card. The lesson is entirely lecture.
- **The Master Taxonomy table overlaps with the 5-1 preview table.** The 5-1 table has 10 rows; the 5-3 table has 19 rows. The overlap is nearly complete -- 5-3 merely expands 5-1. Students encounter the same information twice.
- **Category 5 ("Other Common Flaws") is a catch-all** with 7 sub-types (Ad Hominem, Straw Man, Appeal to Ignorance, False Dichotomy, Circular Reasoning, Equivocation, Appeal to Authority). These are diverse flaws grouped together only because they do not fit the big four categories. Consider whether any deserve their own drill lesson.
- **The LSAT Language Translation Table appears in both 5-3 and 5-12.** The two versions overlap substantially.

**Suggested edits:**
- Add at least one mini-exercise or "identify the flaw" quick check after every 3-4 flaw types to break up the lecture.
- Consider splitting this lesson: "Field Guide Part 1: Causal & Conditional" and "Field Guide Part 2: Statistical, Scope Shift, & Other." Two 60-block lessons are more digestible than one 120-block lesson.
- Remove the LSAT Language Translation Table from this lesson and keep it only in 5-12 (the Reference Guide). Or vice versa, but do not repeat it.
- Remove the flaw category preview table from 5-1 now that the definitive version lives here.

### Lesson 5-4: Drill -- Correlation Does Not Equal Causation (~45 blocks)

**Role:** Focused drill on causal flaws.

**Strengths:**
- Tight conceptual setup: correlation vs. causation defined, three error types in a table, the "Holy Trinity" callout
- Spotting Causal Flaws table (premise language vs. conclusion language) is a powerful pattern-matching tool
- VCR/movie-theater worked example is well-chosen -- the third-factor explanation is intuitive
- Full 5-choice answer breakdown with correct/incorrect badges
- The "Trap Answer Analysis" callout explicitly teaches why a plausible wrong answer fails
- Second practice question (family dinners) with full analysis provides independent practice

**Issues:**
- **No question-card component.** The worked example and practice question are both presented as inline text with process blocks and breakdowns, not as interactive question-cards.
- **The practice question lacks a stimulus blockquote.** The analysis references "Family dinners" but the full stimulus text is missing -- only the analysis appears.
- **Overlap with 5-3.** The Correlation/Causation section of the Field Guide already covers this content. The drill adds the worked examples and practice, which is valuable, but the conceptual setup (blocks 0-12) retreads ground.

**Suggested edits:**
- Add the missing stimulus blockquote for the practice question.
- Use a question-card component instead of inline analysis so students can attempt it first.
- Shorten the conceptual setup to a brief "review the causal flaw from the Field Guide" reference, then jump to the worked example faster.

- **The practice question stimulus is not shown.** Only the analysis appears.
- **The "Critical Question" framing** ("Are these groups of a similar size?") is excellent but could be more prominent -- it is buried in a paragraph rather than being a standalone callout.

**Suggested edits:**
- Add the missing stimulus for the practice question.
- Promote "Are these groups of a similar size?" to a standalone tip callout.
- Use question-card components.

### Lesson 5-8: Advanced Concepts -- Difficult Flaw Questions (~70 blocks)

**Role:** Bridge from drills to advanced material, covering difficulty factors and advanced flaw types.

**Strengths:**
- The three difficulty factors framework (Subtle Flaws, Compound Flaws, Abstract Language) is an excellent metacognitive tool
- The "Causal-Conditional Blur" example (exercise/cardiovascular/longevity) clearly demonstrates how surface language can disguise the underlying flaw type
- The Abstract Flaw Description translation table (14 rows) is the module's most valuable reference for answer-choice decoding
- Three trap answer types (Reverse Logic, Category Mismatch, Unproven Premise) are well-explained with concrete examples
- Equivocation, Circular Reasoning, and False Dichotomy get dedicated sections with examples and LSAT language
- The Elkmount climate/suicide worked example is genuinely hard and teaches the indirect causal chain concept
- The Double-Check Method (5-step verification process) provides a systematic approach for hard questions
- The "(A) is Tempting but Not Primary" analysis explicitly teaches prioritization between multiple real flaws

**Issues:**
- **Covers too many distinct topics.** Three difficulty factors, an abstract language translation table, three trap types, three advanced flaw types, a hard worked example, and the Double-Check Method is at least four lessons worth of content in one.
- **No question-card component.**
- **The abstract language translation table overlaps with the LSAT Language Translation Table in 5-3 and the LSAT Language Decoder in 5-12.** Three versions of essentially the same reference exist.
- **The advanced flaw types (Equivocation, Circular, False Dichotomy) were already introduced in 5-3.** The depth here is only marginally greater.

**Suggested edits:**
- Consider splitting: "Advanced: Difficulty Factors & Traps" and "Advanced: Worked Example & Double-Check Method."

## Structural Issues

### 1. No Question-Cards in Any Lesson

This is the module's most significant structural gap. Despite extensive worked examples, prephrasing snippets, and practice questions, no lesson uses an interactive question-card component. All practice is delivered through inline blockquotes, process blocks, and accordion reveals. Students never experience the format they will encounter on the actual LSAT (stimulus + five answer choices presented together).

**Recommendation:** Add at least one question-card to every drill lesson (5-4 through 5-7) and every advanced drill (5-9 through 5-11). The worked examples already have 5-choice breakdowns -- converting them to question-cards would be a formatting change, not a content change.

### 2. Missing Stimulus Text in Drill Lessons

Lessons 5-4 through 5-11 all have a "Practice Question" section that includes full analysis and answer-choice breakdowns, but several are missing the actual stimulus blockquote. The analysis references the argument, but the student cannot read it independently.

**Recommendation:** Add the missing stimulus blockquotes for every practice question.

### 3. Redundancy Between 5-1, 5-3, 5-8, and 5-12

Four lessons contain flaw taxonomy tables:
- 5-1: Preview table (10 rows, ~frequencies)
- 5-3: Master Taxonomy (19 rows, ~frequencies)
- 5-8: Abstract Language Translation Table (14 rows)
- 5-12: Complete Taxonomy + LSAT Decoder (20 rows + 20 rows)

Additionally, three lessons contain a Quick Identification Flowchart / Mental Flaw Checklist (5-2, 5-3, 5-12). Students encounter substantially similar reference material four times.

**Recommendation:** Designate 5-12 as the canonical reference. Cut the preview table from 5-1 (replace with a forward reference). Cut the LSAT Language Translation Table from 5-3 (keep only in 5-12). Cut the abstract language table from 5-8 (reference 5-12). Keep the Mental Flaw Checklist in 5-2 (where students first learn it) and 5-12 (the reference), but remove it from 5-3.

### 4. Lesson 5-3 Length

At ~120 blocks, the Field Guide is almost certainly the longest lesson in the course. It covers 19 flaw types without any practice breaks. A student reading linearly will encounter 5+ categories of conceptual content before getting any chance to apply it.

**Recommendation:** Split into two lessons (Causal & Conditional; Statistical, Scope Shift, & Other) or add mini-exercises after every 3-4 flaw types.

## Summary of Priorities

1. **Add question-cards throughout** -- the module's biggest structural gap
2. **Add missing stimulus blockquotes** -- several practice questions are headless
3. **Eliminate taxonomy/reference table redundancy** across 5-1, 5-3, 5-8, and 5-12
4. **Split or add exercises to Lesson 5-3** -- 120 blocks of lecture with zero practice
5. **Add "try it first" prompts** before every worked analysis -- small change, big pedagogical impact
