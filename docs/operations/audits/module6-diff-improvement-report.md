# Module 6: Parallel Flaw -- Website Content Improvement Report

**Scope:** The 10 website lessons (6-1 through 6-10) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 6 is an exceptionally well-structured unit that treats what many students consider the hardest LR question type with systematic precision. The 10 lessons progress from introduction through six targeted drill lessons, an advanced lesson, and a complete reference guide. Every drill lesson (6-3 through 6-9) includes a real PT question card with full wrong-answer analysis. The primary improvement opportunities are:

1. **Lesson 6-1 tries to do too much** -- it covers introduction, comparison to Parallel Reasoning, the Two-Step Process, the full flaw types table, a worked example, AND a difficulty warning. This is 41 blocks serving as both overview and comprehensive content reference. The Parallel Flaw vs. Parallel Reasoning comparison table and the flaw types table both reappear in later lessons.
2. **No practice question in the first 2 lessons** -- the student reads ~93 blocks of theory (6-1 + 6-2) before encountering the first question card in Lesson 6-3. The worked example in 6-1 uses a blockquote, not an interactive question card.
3. **Lesson 6-2 lacks a practice question** -- this is the method lesson (3-Step Method) but contains only a hypothetical walkthrough about parks/crime, not an actual question card the student can attempt.
4. **Lesson 6-4 is missing its question card** -- the analysis section references a Historian argument about democracy/monarchy (Composition flaw), but the question-card block itself is absent. The analysis starts without the student seeing the stimulus.
5. **The drill lessons (6-3 through 6-9) are excellent** -- each has a focused flaw concept, a real PT question, comprehensive wrong-answer breakdown, and a trap analysis section. These are the gold standard.
6. **The reference guide (6-10) is comprehensive** -- at 31 blocks it covers 11 flaw types, the full matching checklist, 7 trap types, timing strategy, and a test-day checklist. This is significantly richer than Module 1's reference guide.

## Lesson-by-Lesson Review

### Lesson 6-1: Introduction (41 blocks)

**Role:** Module introduction and overview.

**Strengths:**
- Clear Question Goal statement up front
- The "Why This Matters" section effectively positions Parallel Flaw as the synthesis of Modules 4 and 5
- The Parallel Flaw vs. Parallel Reasoning comparison table is excellent -- addresses the most common student confusion directly
- Question stem recognition table with 5 patterns is comprehensive and accurate
- The Frequency & Timing callout (1 per section, 2:00-2:30 budget) sets realistic expectations
- The Common Flaw Types table with frequency ratings (star system) gives students a prioritization framework
- The "Big Four" callout is the single most actionable insight in the lesson

**Issues:**
- **Scope creep:** The lesson covers introduction, Parallel Flaw vs. Parallel Reasoning comparison, the Two-Step Process, the full flaw types table, a worked example, AND a difficulty warning. That is six distinct teaching goals in one lesson.
- **The Parallel Flaw vs. Parallel Reasoning table (blocks 14-17) duplicates the Reference Guide (6-10).** Both have the same 7-row comparison. One should be the canonical version.
- **The flaw types table (blocks 22-27) also appears in the Reference Guide.** The 6-10 version is richer (includes LSAT language column). The 6-1 version could be reduced to the "Big Four" only and reference 6-10 for the complete table.
- **No practice question.** The worked example (bestseller list) uses a blockquote + walkthrough, not an interactive question card. The student reads 41 blocks of theory without attempting anything.
- **The "Time Management Warning" callout (block 36) is very long** -- it contains 3 bullet points plus a strategy recommendation. This could be a separate process block for better scannability.

**Suggested edits:**
- Cut the Parallel Flaw vs. Parallel Reasoning table. Replace with a brief 2-sentence summary and a forward reference to the Reference Guide.
- Reduce the flaw types table to the "Big Four" (Mistaken Reversal, Mistaken Negation, Composition/Division, Causal Confusion). Point to 6-10 for the complete table.
- Add a lightweight exercise after the bestseller-list walkthrough: show a short argument and ask the student to name the flaw before revealing the answer.

### Lesson 6-2: The 3-Step Method (52 blocks)

**Role:** The complete method for solving Parallel Flaw questions.

**Strengths:**
- Excellent structure: overview process block, then each step gets its own H2 section with detailed guidance.
- The breakdown table for "If the Stimulus Does This..." is one of the strongest diagnostic tools in the module -- 8 flaw patterns with badges and colors.
- Blueprint Examples table (7 flaw types mapped to abstract patterns) is highly useful and well-formatted.
- The Flaw-First Strategy callout quantifies the time savings (40 sec investment saves 75+ sec) -- concrete and persuasive.
- The Flaw Matching Checklist (4 checks: flawed? same type? same direction? same force?) is the operational backbone of the module.
- Common Wrong Answer Patterns table (5 patterns) is practical and well-explained.
- The "Worked Method Snapshot" section demonstrates the method on a causal-confusion example without needing a full question card.

**Issues:**
- **No question card.** Despite being the method lesson, there is no interactive practice question. The "Worked Method Snapshot" is a hypothetical about parks/crime, not a real LSAT question the student can attempt.
- **Length.** At 52 blocks, this is the longest lesson in the module. Steps 1-3 are each thorough, but the combined reading may cause students to disengage before reaching the worked snapshot.
- **The breakdown diagnostic table (8 items) repeats information from 6-1's flaw types table.** The format is different (breakdown vs. table) but the content overlaps significantly.
- **The "Quick Reference: The Steps" section at the end** re-states the 3-step method that was already given in the Overview at the top. This is a minor redundancy but adds to the length.

**Suggested edits:**
- Add a question card after the Worked Method Snapshot. A causal-confusion PT question would align with the snapshot and give students immediate practice applying all 3 steps.
- Consider whether the Step 1 breakdown diagnostic table can reference 6-1's flaw types table rather than restating it. The breakdown format adds value (badges, colors), but the content is largely the same.
- Cut the "Quick Reference: The Steps" section at the bottom -- it is redundant with the Overview at the top.

### Lesson 6-3: Drill -- Mistaken Negation (43 blocks)

**Role:** Focused concept lesson + practice on Mistaken Negation.

**Strengths:**
- Clear conceptual breakdown: rule, observation, flawed conclusion, each with badges.
- The Valid vs. Invalid Conditional Inferences table (4 rows) is the definitive reference for conditional logic on the LSAT -- covers Modus Ponens, Modus Tollens, Mistaken Reversal, and Mistaken Negation.
- The Memory Aid callout explains valid vs. invalid in terms of "staying on the same side of the arrow."
- Mistaken Negation vs. Mistaken Reversal comparison table addresses the primary confusion directly.
- Real question card: PT-106-S-3-Q-21 (medium difficulty).
- Full 5-choice breakdown with badges (Correct / Incorrect + flaw name for each).
- The trap analysis identifies the Adjacent Flaw Trap (A, C) and the Valid Logic Trap (B, E) -- the two main trap categories for conditional questions.

**Issues:**
- **No "try it yourself" prompt before the analysis.** The question card is immediately followed by the full step-by-step analysis. A pause would improve active learning.
- **The Valid vs. Invalid table could reference Module 3** (if conditional logic was covered there) to reduce the sense of encountering this material for the first time.

**Suggested edits:**
- Add a brief "Attempt this question before reading the analysis below" prompt between the question card and the analysis.
- Minor: the formal notation ($\rightarrow$) in the breakdown appears to use LaTeX-style syntax. Verify this renders correctly in the ContentBlockRenderer.

### Lesson 6-6: Advanced -- Deconstructing High-Difficulty (54 blocks)

**Role:** Bridge from basic to advanced, covering three difficulty factors.

**Strengths:**
- Clearly identifies three difficulty factors: Compound Flaws, Abstract Structure, and Multiple Viable Candidates. This is the right taxonomy.
- The compound flaw example (bank deposit + Alicia) is reused in 6-7, creating coherent cross-lesson progression.
- The Structural Elimination Method (5 steps) extends the standard 3-step method with explicit checkpoints for hard questions.
- The 6-column advanced matching dimensions table is the most granular matching tool in the module.
- The 5 advanced trap types (Half-Match, Topic Twin, Adjacent Flaw, Valid Decoy, Overcomplicated Choice) are well-named and well-explained.
- The Skip-and-Return Strategy and 90-Second Rule are practical time management advice.

**Issues:**
- **Same missing-card pattern.** The "Practice Question" H2 + hr appears, but no question-card block. The analysis references Riaz/endorsements/campaign funding but the student never sees the full question.
- **Overlap with 6-7.** The bank deposit/Alicia example in the "Compound Flaws" section is the same stimulus used as the practice question in 6-7 (Double Flaw drill). Students may feel the advanced lesson spoils the drill lesson.
- **54 blocks is very long.** Three difficulty factors, a 5-step method, a 6-column matching table, 5 trap types, and a full worked example make this a dense lesson.

**Suggested edits:**
- **Critical:** Restore the missing question card.
- Consider whether the bank deposit example should be kept here or saved for 6-7 to avoid spoiling the drill. If kept, note the connection explicitly.
- The lesson could be split: Factor 1 (Compound Flaws) in one lesson, Factors 2-3 (Abstract Structure + Multiple Candidates) in another, since Factor 1 is substantially different from the other two.

## Structural Issues

### 1. Three Missing Question Cards (6-4, 6-5, 6-6)

Lessons 6-4, 6-5, and 6-6 each have a "Practice Question" heading followed by an hr, then jump directly to the step-by-step analysis. The question-card blocks containing the stimulus, question stem, and answer choices are absent. The analyses reference specific arguments (Historian/democracy for 6-4, vegetarian/heart-disease for 6-5, Riaz/endorsements for 6-6) that the student never sees. This is the module's most critical content gap.

**Recommendation:** Restore the three missing question cards. Each lesson's analysis section is complete and correct -- only the question-card block is missing from the content array.

### 2. Practice Drought in Lessons 6-1 and 6-2

The first two lessons contain ~93 blocks of theory and zero practice questions. The first question card appears in Lesson 6-3. The worked example in 6-1 (bestseller list) uses a blockquote, not a question card. The worked snapshot in 6-2 (parks/crime) is hypothetical.

**Recommendation:** Add at least one lightweight exercise to each of the first two lessons. Even "Name the flaw in this 2-sentence argument" before revealing the answer would help.

### 3. Overlap Between 6-6 and 6-7

The bank deposit/Alicia stimulus appears as an illustrative example in the Advanced lesson (6-6) and as the practice question in the Double Flaw drill (6-7). Students who read sequentially encounter the same argument twice, and the advanced lesson spoils the drill.

**Recommendation:** Use a different compound-flaw example in 6-6 to preserve the PT-113-S-3-Q-21 question for 6-7.

### 4. Redundant Content Between 6-1 and 6-10

The Parallel Flaw vs. Parallel Reasoning comparison table appears in both 6-1 and 6-10 with identical content. The flaw types table appears in 6-1 (with frequency stars) and 6-10 (with LSAT language column) in different formats. Neither lesson references the other.

**Recommendation:** Lesson 6-1 should use condensed versions (Big Four only for flaw types, 2-sentence summary for PF vs. PR) and point to 6-10 as the complete reference. Or: merge the two tables (frequency + LSAT language) in 6-10 and remove the full version from 6-1.
