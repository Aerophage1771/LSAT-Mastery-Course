# Module 19: Paradox / Explain -- Website Content Improvement Report

**Scope:** The 10 website lessons (19-1 through 19-10) evaluated as standalone content.
**Date:** 2026-04-01

---

## Executive Summary

Module 19 is one of the strongest modules in the course. The 10 lessons cover introduction through an advanced EXCEPT drill and a comprehensive reference guide, with six real PT question cards (Lessons 3--5, 7--9) and full worked analyses. The pedagogical arc is clear: concept introduction, method, five paradox sub-type drills, an advanced-difficulty guide, and a reference consolidation. The primary improvement opportunities are:

1. **Missing "try it first" prompts before worked analyses** -- Lessons 3--5, 7--9 all include question cards with analyses that follow immediately. Only Lesson 4 has an explicit "commit to an answer before scrolling" instruction in the body text; Lessons 3, 5, 7, and 8 place the question card directly before the analysis with no pause prompt.
2. **Lesson 6 (Hardest Paradox) references a question but has no question-card** -- The analysis discusses a parking-fines paradox with full option evaluation, but no question-card block exists for the student to attempt first. The practice section contains only `<br/>` spacers.
3. **Module numbering is inconsistent** -- Lesson 1 header says "MODULE 20" while the module directory is `module19`. Lesson 10 also says "Module 20." This numbering mismatch will confuse students.
4. **The reference guide is the module's crown jewel** -- At 7 major sections with comprehensive tables, trap catalogs, and the EXCEPT strategy, it is the most thorough reference guide in the course. Minor additions would make it definitive.
5. **The "glance" and "playbook" alternates on Lessons 1--2 contain leftover Must Be False content** -- These describe a completely different question type and should be replaced or removed.

## Lesson-by-Lesson Review

### Lesson 19-1: Introduction (14 content blocks)

**Role:** Module overview, question type orientation, core concepts.

**Strengths:**
- Clean, concise introduction that immediately differentiates Paradox from argument-based question types
- The question-stem recognition table is practical and covers the three main stem variants
- "Paradox vs. Strengthen" section directly addresses the most common student confusion
- "The Two-Fact Structure" section is well-framed with useful pivot-word list
- The Common Paradox Types list previews the drill lessons ahead
- The Concept Snapshot (streetlights example) is intuitive and memorable

**Issues:**
- **No practice question.** The lesson is pure lecture. Even a lightweight "identify the two facts" exercise would improve retention.
- **No "try it first" prompt.** Not applicable since there is no question, but the Concept Snapshot could be posed as a mini-challenge before revealing the resolution.
- **The Common Paradox Types list is somewhat generic.** The labels (Numerical, Causal, Behavioral, Temporal, Definitional) do not match the drill lesson titles (Numbers vs. Percentages, Selection Bias, Causal Mechanisms, Hidden Distinction, Counterintuitive Causal Web). Aligning terminology would improve cross-lesson navigation.

**Suggested edits:**
- Add a mini-exercise after the Concept Snapshot: "Before reading on, what kind of new fact would explain this?"
- Align the paradox-type names to match the drill lesson titles.

### Lesson 19-2: Step-by-Step Guide (9 content blocks)

**Role:** The complete method for solving Paradox questions.

**Strengths:**
- Tight and focused at 9 blocks -- no bloat
- The 4-Step Method process block is clear and actionable
- Wrong Answer Patterns list covers the three most important traps
- Worked Method Snapshot (bonus program) is compact and illustrative
- Timing Tip callout gives practical advice

**Issues:**
- **No practice question.** The method lesson has no question-card. The Worked Method Snapshot walks through an analysis but gives no opportunity for the student to attempt a problem.
- **No "try it first" prompt.** The snapshot reveals the analysis inline with no pause.
- **The wrong-answer patterns list has only 3 items.** The reference guide (19-10) identifies 7 traps. Adding "Restates the Paradox" and "Explains the Wrong Direction" here would better prepare students for the drills.

**Suggested edits:**
- Add a lightweight question-card after the method description so students can apply the 4-step method immediately.
- Expand the wrong-answer patterns to at least 5 items.
- Add "Attempt the question before reading the analysis" before the Worked Method Snapshot.

### Lesson 19-3: Drill -- Numbers vs. Percentages (19 content blocks)

**Role:** First drill lesson; teaches the numbers-vs-percentages paradox pattern.

**Strengths:**
- Excellent conceptual setup: the three-variant list (percentage up/number down, percentage down/number up, subgroup vs. whole) is comprehensive
- Recognition Cues table is practical and scannable

**Issues:**
- **No "try it first" prompt.** The question-card is followed immediately by the Full Analysis section. There is no explicit instruction to attempt the question before reading.
- **The "How It Works" section could benefit from a brief numerical example** showing how percentages and absolute numbers diverge, rather than relying entirely on verbal description.

**Suggested edits:**
- Add "Try this: Attempt the question before reading the analysis below." between the question-card and the Full Analysis heading.
- Consider a brief numerical mini-example (e.g., "If 10 out of 100 people = 10%, but then 8 out of 50 people = 16% -- the rate went up but the count went down").

## Structural Issues

### 1. Module Number Mismatch

The mastery course directory is `module19` and lesson IDs are `19-1` through `19-10`, but multiple lesson headers and body text refer to "Module 20." This should be corrected to "Module 19" throughout.

### 2. Missing "Try It First" Prompts (HIGH PRIORITY)

Six lessons (3, 5, 6, 7, 8, 9) include worked analyses for real PT questions but lack an explicit "attempt the question before reading" prompt. Only Lesson 4 includes this instruction in body text. This is the module's single biggest pedagogical gap.

**Recommendation:** Add `**Try this:** Attempt the question before reading the analysis below.` as a paragraph or callout immediately before each analysis section.

### 3. Lesson 6 Missing Question-Card

The Hardest Paradox lesson discusses a parking-fines problem with full option evaluation but has no `question-card` block. Students cannot see the stimulus or attempt the question independently.

**Recommendation:** Add a `question-card` block with the parking-fines stimulus and options before the Resolution & Analysis section.

### 4. Stale Alternate Content on Lessons 1--2

The `glance` alternate on Lesson 1 and the `playbook` alternate on Lesson 2 contain Must Be False content that belongs to a different module entirely. These should be replaced with Paradox-appropriate content or removed.

### 5. Lesson 9 Title Mismatch

The file is named `Lesson9_Drill_FlawedLogicExcept` and the lesson title says "Drill: Flawed Logic in EXCEPT Questions," but the actual content is about Paradox EXCEPT questions. The body heading correctly says "Drill: Explain EXCEPT."

**Recommendation:** Rename to match the body content.
