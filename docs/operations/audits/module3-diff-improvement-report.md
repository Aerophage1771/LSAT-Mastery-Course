# Module 3: Method of Reasoning -- Website Content Improvement Report

**Scope:** The 12 website lessons (3-1 through 3-12) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 3 is a well-structured and pedagogically strong unit. The 12 lessons progress cleanly from introduction through advanced concepts, and lessons 3-4 through 3-11 each include a real PT question-card with full worked analysis and wrong-answer breakdowns. The primary improvement opportunities are:

1. **The Introduction (3-1) tries to serve as both overview and mini field guide** -- it covers strategy vs. content, method vs. argument part, all 8 method categories, indicator words, AND a walkthrough. At ~45 blocks this is a dense opener with no practice question.
2. **Lesson 3-3 (Field Guide) heavily overlaps 3-1** -- both present the same method category table with nearly identical frequencies, descriptions, and indicator language. The student reads the same catalog twice.
3. **No practice questions in the first 3 lessons** -- the student encounters ~180 blocks of pure theory before the first question-card in Lesson 3-4. The opening lessons are all lecture with no hands-on engagement.
4. **Lessons 3-4 through 3-11 are excellent** -- each has a focused concept, a real PT question, and a full wrong-answer analysis. This is the gold standard the earlier lessons should match.
5. **Lesson 3-8 (Advanced Concepts) has no question-card** -- it describes hard-question factors and has a partial worked example but no interactive practice, making it the weakest link in the advanced sequence.
6. **The Reference Guide (3-12) is comprehensive** -- at ~70+ blocks it is far richer than Module 1's reference. It includes a decoder table, trap breakdown, indicator words, and a test-day checklist. The main gap is the absence of a prephrase-examples table and the Abstract Translation Technique from 3-2.

## Lesson-by-Lesson Review

### Lesson 3-1: MODULE 3: Method of Reasoning (~45 blocks)

**Role:** Module introduction and overview.

**Strengths:**
- Clear Question Goal statement up front
- The "Why This Matters" section effectively motivates the topic with a concrete dependency chain (Flaw, Parallel Reasoning, Weaken/Strengthen)
- The Frequency & Difficulty section gives helpful context (1-2 per section, difficulty from translation not reasoning)
- The Question Stem Recognition table is excellent -- five distinct patterns with key signals
- The "Strategy vs. Content" section with the car/alternator breakdown is the single best teaching moment in the lesson
- The "Abstract Description Technique" callout is actionable and well-phrased
- The Method vs. Argument Part comparison table is a strong differentiator from Module 1
- Common Method Categories preview table with frequencies and indicator language is useful

**Issues:**
- **Scope creep.** The lesson covers: introduction, strategy vs. content, method vs. argument part, all 8 method categories with frequencies and indicators, a full indicator phrases table, AND a first walkthrough. That is six distinct teaching goals in one lesson.
- **The Method Categories table (8 rows with frequencies, descriptions, and indicator language) duplicates Lesson 3-3** almost exactly. The field guide repeats the same catalog with the same approximate percentages and the same indicator phrases.
- **The Indicator Words table duplicates 3-3's indicator language sections.** Both lessons map the same phrases to the same methods.
- **No practice question.** The student reads ~45 blocks of theory without ever being asked to try anything. The "Your First Method of Reasoning Question" section provides a walkthrough but no actual question-card for the student to attempt independently.
- **The walkthrough uses a paraphrased example (meteorologist/water shortage) rather than a real question-card.** Students see the method described but never practice the strategy-to-content translation themselves.

**Suggested edits:**
- Cut the Method Categories table and Indicator Words table down to a brief 3-row preview (top 3 methods only) and point to 3-3 for the full catalog.
- Add a mini-exercise or lightweight question-card after the car/alternator breakdown.
- Convert the meteorologist walkthrough into an actual question-card the student can attempt before seeing the analysis.

### Lesson 3-2: The 4-Step Method (~70 blocks)

**Role:** The complete repeatable method for solving Method of Reasoning questions.

**Strengths:**
- Excellent structure: overview process block, then each step gets its own H2 section with example, tip callout, and "when it gets tricky" callout.
- The timing callout (1:30 per question) is valuable and unique to this lesson.
- The Abstract Translation Technique table (4 rows mapping concrete to abstract) is one of the strongest teaching tools in the module.
- The Prephrase Checklist callout (action, relationship, goal) is actionable and memorable.
- The Common Method Quick-Reference list is a useful rapid-check tool during practice.
- Step 4's Three Elimination Checks (Action, Scope, Speaker) are clear, distinct, and correct.
- The Common Wrong Answer Patterns table (5 trap types) is well-structured and LSAT-accurate.

**Issues:**
- **Length.** At ~70 blocks, this is one of the longest lessons in the module. Steps 1-4 are each thorough, but the total reading time may cause focus loss before the worked example.
- **No question-card.** Despite being the method lesson, there is no interactive practice. The worked example (nutritionist/diet book) is a process walkthrough, not a question-card the student can attempt before seeing the answer.
- **The Rome example in Step 1 and the nutritionist example in the Worked Example are disconnected.** Using the same argument across all 4 steps would create a running thread the student follows from start to finish.
- **The "When This Step Gets Tricky" callouts are individually great but collectively add bulk.** Consider moving some edge-case detail to the Reference Guide.

**Suggested edits:**
- Add a question-card after the Full Worked Example section, giving the student one chance to apply the method independently.
- Consider using the same argument across all 4 steps instead of switching between the historian, environmentalist, detective, and nutritionist.
- Tighten Steps 1 and 2 -- they share overlapping advice about identifying argument components.

### Lesson 3-3: A Field Guide to Common Strategies (~85 blocks)

**Role:** Detailed method catalog with frequency data, variations, examples, and indicator language.

**Strengths:**
- The Master Summary Table (9 rows, 4 columns) is an excellent quick reference -- frequencies, core descriptions, and answer choice language in one table.
- Each of the 7 method sections gets a deep dive: concept, variations, concrete example, and indicator language.
- The analogy section correctly distinguishes three sub-types: supportive, refutation, and attacking an analogy.
- The Proof by Contradiction variation is correctly placed under Consequences (not as a standalone method at this stage).
- The Quick Pattern Matching decision process (8-step flowchart) is highly actionable for timed practice.
- The Method Indicator Language quick reference (mapping answer choice language to methods) is a unique and useful tool.

**Issues:**
- **Heavy overlap with 3-1.** The Master Summary Table repeats the same 8 categories, same approximate frequencies, and same indicator language as 3-1's "Common Method Categories" table. A student who reads both encounters the same catalog twice.
- **85 blocks is very long** for a single lesson. The combination of master table + 7 deep dives + pattern matching process + indicator reference is three distinct lessons packed into one.
- **No question-card.** This lesson introduces all 7+ methods with concrete examples but no interactive practice. At minimum, the Quick Pattern Matching section should include a short exercise.
- **The "Other" category (~12%) in the Master Summary Table is vague.** It lists "Reductio, definition disputes, conditional logic" without further explanation. Students may not know what these are until the advanced lessons.

**Suggested edits:**
- If 3-1 is trimmed (see above), this lesson becomes the definitive method catalog -- no overlap.
- Add one question-card or mini-exercise, ideally after the Quick Pattern Matching section.
- Consider splitting: Method Catalog (3-3a) and Pattern Matching / Indicator Reference (3-3b) if length is a concern.
- Expand the "Other" row with brief one-sentence descriptions of reductio, definition disputes, and conditional logic.

## Structural Issues

### 1. Redundancy Between Lessons 3-1 and 3-3

The 8 method categories are taught in full twice: once in 3-1 (Common Method Categories table + Indicator Words table) and again in 3-3 (Master Summary Table + 7 deep-dive sections + Indicator Language reference). Both present the same approximate frequencies, the same descriptions, and the same indicator phrases. This is the module's biggest structural problem.

**Recommendation:** Lesson 3-1 should present only a brief preview (top 3 methods as a teaser) and point to 3-3 for the complete catalog. Or: 3-1 covers the method concept with one worked example, and 3-3 owns the full taxonomy.

### 2. Practice Drought in Lessons 3-1 through 3-3

The first three lessons contain ~180 blocks of theory and zero practice questions. The first question-card appears in Lesson 3-4. Students who learn best by doing will disengage before they get a chance to try anything.

**Recommendation:** Add at least one lightweight exercise (not necessarily a full question-card) to each of the first three lessons. Even "identify the method in this 3-sentence argument" in a callout would help.

### 3. Lesson 3-8 Has an Incomplete Worked Example

Lesson 3-8 (Advanced Concepts) has an analysis section that references "Professor Wexler" and carbon tax economists, with wrong-answer analysis, but the question-card may be missing or incomplete. The student sees the reasoning analysis but cannot attempt the question independently.

**Recommendation:** Add the complete question-card for the Wexler/carbon tax example. This is the only advanced lesson without a proper interactive practice opportunity.

### 4. No "Try It First" Prompts Anywhere

None of the 8 question-card lessons (3-4 through 3-11) includes a prompt encouraging students to attempt the question before reading the analysis. The analysis follows the question-card immediately, making it easy to passively read through rather than actively practice.

**Recommendation:** Add a brief "Attempt this question before reading the analysis below" prompt before each analysis section. Small change, big pedagogical impact.
