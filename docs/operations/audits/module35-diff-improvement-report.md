# Module 35: Phrase Function -- Website Content Improvement Report

**Scope:** The 2 website lessons (35-1 through 35-2) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 35 covers Phrase Function questions -- one of the highest-frequency RC question types at 152 appearances (over 7%). The two lessons provide a clear introduction with stem identification and a solid step-by-step approach. The module correctly distinguishes Phrase Function from Phrase Meaning (M34), focusing on "why the author included it" rather than "what it means." Key findings:

1. **Zero practice questions** -- neither lesson includes a question-card or a complete worked example. For the second-most-common RC question type, this is a significant gap.
2. **The "what vs. why" distinction is the module's best teaching moment** -- the callout in 35-1 and the elimination note in 35-2 both reinforce this critical insight effectively.
3. **Good overlap awareness between M34 and M35** -- the lesson implicitly distinguishes meaning from function, but an explicit comparison table would help students who study these modules back-to-back.
4. **The five-item role list in 35-2 is strong** -- Evidence, Example, Counterargument, Definition, Cause/Effect covers the most common functional roles well.

## Lesson-by-Lesson Review

### Lesson 35-1: Germaine's Guide to RC Phrase Function Questions (~15 blocks)

**Role:** Module introduction, concept overview, and stem identification guide.

**Strengths:**
- Clear opening that correctly frames the task as identifying purpose, not meaning
- The four-item skill list (rhetorical role, surrounding context, main argument connection, focus on intent) effectively separates Phrase Function from Phrase Meaning
- The stem identification breakdown covers three categories (Purpose-Driven, Functional, Exemplification) with real stem language
- The Overarching Takeaway callout is precise and memorable: "not what the detail says, but why the author included it"
- Frequency data (152 appearances, 7%) correctly positions this as a high-priority question type
- Cross-references to Paragraph Function, Structure, and Main Point questions show how this skill transfers

**Issues:**
- **No practice question.** For a question type appearing 152 times, the absence of any practice is notable.
- **No explicit comparison to Phrase Meaning (M34).** Since students will likely study these modules in sequence, a brief "Phrase Function vs. Phrase Meaning" table would prevent confusion.
- **The three stem categories could use more differentiation.** Purpose-Driven and Functional phrasing overlap significantly ("in order to" vs. "serves to"). A note explaining the subtle difference would help.
- **No common wrong-answer patterns.** The lesson does not warn about the most frequent trap: answers that correctly describe what the phrase says rather than why it is there.

**Suggested edits:**
- Add a "Phrase Function vs. Phrase Meaning" comparison table (2-3 rows showing the same stem phrased both ways).
- Add a "Common Trap" callout about "what it says" vs. "why it's there" wrong answers.
- Add a brief stem identification exercise.

---

### Lesson 35-2: Step-by-Step Approach (~15 blocks)

**Role:** The complete method for solving Phrase Function questions.

**Strengths:**
- Clean three-step method: Locate + Read Context, Determine Relationship to Local Point, Test Against Prediction
- The five-item functional role list (Evidence, Example, Counterargument, Definition, Cause/Effect) is comprehensive and practical
- Step 2's framing ("How does this detail relate to the sentence right before it?") correctly anchors the analysis in local context
- The explicit elimination note -- "Eliminate answer choices that accurately describe what the phrase says but fail to describe why it's there" -- is the lesson's strongest teaching moment
- Two tie-breakers ("In Order To" Test, Zoom Out to Paragraph Purpose) address both local and global analysis

**Issues:**
- **No question-card.** The method is explained well but never practiced.
- **No worked example.** Unlike some other modules, this lesson does not include even a hypothetical walkthrough demonstrating the method in action.
- **The "In Order To" Test tie-breaker is excellent but untested.** It describes a powerful reframing technique but does not show it applied to competing answer choices.
- **No warning about the "too global" trap.** Function questions target a specific phrase, and students sometimes select answers that describe the function of the entire paragraph rather than the specific detail.

**Suggested edits:**
- Add a worked example demonstrating the "In Order To" Test on two competing answer choices.
- Add a question-card after the method walkthrough.
- Add a brief "too global" trap warning: answers that describe the paragraph's function rather than the specific phrase's function.

## Structural Issues

### 1. No Practice for a High-Frequency Question Type

At 152 appearances, Phrase Function is one of the most common RC question types. The module provides zero practice. This is disproportionate to the question type's importance.

**Recommendation:** Add at least one question-card to 35-2, and ideally add a brief exercise to 35-1 as well.

### 2. M34/M35 Confusion Risk

Students studying Phrase Meaning (M34) and Phrase Function (M35) back-to-back may confuse the two question types. Neither module explicitly differentiates from the other.

**Recommendation:** Add a brief comparison table to 35-1 showing how the same phrase could be tested as either Meaning or Function, with different stems and different correct answers.
