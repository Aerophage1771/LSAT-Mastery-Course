# Module 57: Advanced Passage 7 -- Digitalization and Copyright Law -- Website Content Improvement Report

**Scope:** The 6 website lessons (57-1 through 57-6) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 57 covers a single RC passage about Canadian copyright law and digitalization, with one passage analysis lesson and five question breakdown lessons (Q24--Q28). The passage is a law-focused policy argument written in the mid-1990s -- a classic LSAT topic that tests the student's ability to track a problem-proposal-complications structure. The lesson quality is consistent and the analysis work is strong. The primary improvement opportunities are:

1. **No "try it yourself" prompts before any worked analysis** -- the same gap found in M55 and M56. All five question lessons jump from the answer choices directly to the analysis.
2. **The passage analysis (57-1) is the weakest of the three modules** -- it has only 6 blocks (compared to 8 in M55 and 10 in M56) and lacks an HR separator or visual structure between the passage and the analysis.
3. **Strong question quality and analysis** -- the five questions cover Main Point (Q24), Compromise/Application (Q25), Function (Q26), EXCEPT (Q27), and Author Viewpoint/Inference (Q28). Each analysis is well-reasoned.
4. **The passage includes a temporal context note** ("The following passage was written in the mid-1990s") that none of the lessons address. This is a pedagogical opportunity -- students should learn to note these context markers.
5. **Q25 (Compromise question) is an unusual question type** that deserves explicit methodology teaching. It requires the student to synthesize both parties' positions and find a middle ground, which is different from standard inference or application questions.

## Lesson-by-Lesson Review

### Lesson 57-1: Passage Analysis (6 blocks)

**Role:** Passage presentation with paragraph-by-paragraph breakdown.

**Strengths:**
- Each paragraph gets its own H4 heading with a clear, descriptive nickname ("The Problem," "Why the Law is Outdated," "Why Fixing the Law is Hard").
- The Summary callout at the end is well-structured and captures the three-layer argument (problem, legal loophole, practical complications).
- The analysis uses strong, specific language -- "legal loophole," "philosophical fight," "information wants to be free" -- that helps students remember key concepts.

**Issues:**
- **Thinnest passage analysis of the three modules.** At only 6 blocks, it lacks the visual structure (HR separator) and depth of M55 (8 blocks) and M56 (10 blocks).
- **No approach tips.** Like M55 and M56, this lesson does not teach reading strategy. For a law/policy passage, noting the problem-proposal-complications structure would be especially valuable.
- **The temporal context note is ignored.** The passage begins with "The following passage was written in the mid-1990s." This is a significant detail for a passage about Internet copyright -- the technology context has changed dramatically. None of the lessons mention why this matters or how to handle temporal context notes on the LSAT.
- **No mention of the three specific complications** raised in P3 (defining the crime, enforcement, academic exemptions). These are the most testable details in the passage.

**Suggested edits:**
- Add a "Reading Strategy" callout noting: (1) this is a problem-proposal-complications passage -- mark the shift from P2's legal analysis to P3's objections, (2) P3 contains three distinct complications -- number them as you read, (3) the passage ends with a "deeper conflict" that the author believes will persist even after legal reform.
- Add a note about the temporal context: "The 'mid-1990s' note tells you the passage was written when the Internet was still new. On the LSAT, temporal context notes signal that the issues discussed may have evolved, but you should answer based only on the passage as written."
- Add an HR separator between the passage accordion and the paragraph analysis for visual consistency with M56.
- Enumerate the three complications in the P3 analysis.

### Lesson 57-3: Question 25 Breakdown (7 blocks)

**Role:** Compromise / Application question.

**Strengths:**
- The Prediction callout correctly identifies the core interests of both parties: internet users want access, publishers want compensation. A compromise must address both.
- Excellent analysis of (A) -- correctly identifies that paying a fee satisfies both groups.
- Clean elimination of (C) and (D) -- correctly identifies them as one-sided solutions.
- Good analysis of (E) -- correctly notes that the academic exemption already exists under current law.

**Issues:**
- **No "try it yourself" prompt** before the analysis section.
- **Compromise questions are unusual on the LSAT.** A brief methodology note about how to approach them (identify each party's core interest, then find the answer that partially satisfies both) would be valuable.
- **Answer (A) has a 65% correct rate**, with (E) as the main distractor at 23%. The lesson could note why (E) is tempting but wrong.

**Suggested edits:**
- Add the standard "Try this" prompt.
- Add a methodology callout: "Compromise questions ask you to find a solution that gives something to both sides. Strategy: (1) Identify what each party wants. (2) Eliminate answers that only satisfy one side. (3) The correct answer usually involves a trade-off -- one side gets access, the other gets compensation or protection."
- Add a brief note on why (E) is a trap: "Answer (E) is tempting because it mentions research, which the passage discusses at length. But it does not represent a compromise -- it merely restates an exemption that already exists under current law."

## Structural Issues

### 1. No Active Practice Moment in Any Question Lesson

All five question lessons (57-2 through 57-6) present the question and immediately provide the full analysis. This is the same gap found in M55 and M56.

**Recommendation:** Add a "Try this: Attempt the question before reading the analysis below" callout between the options block and the analysis paragraph in every question lesson.

### 2. Passage Analysis Is the Thinnest of the Three Modules

Lesson 57-1 has only 6 blocks -- no HR separator, no approach tips, and no enumeration of the three complications in P3. It is the least developed passage analysis in the advanced passage set.

**Recommendation:** Expand 57-1 with a reading strategy section, an HR separator, a note about the temporal context, and explicit enumeration of P3's three complications.

### 3. Temporal Context Note Is Unused

The passage begins with "The following passage was written in the mid-1990s" -- a significant detail that none of the lessons address. This is a missed teaching opportunity.

**Recommendation:** Add a brief note to 57-1 explaining how to handle temporal context markers on the LSAT.

### 4. Q25 (Compromise) and Q28 (Expert Viewpoint) Are Unusual Question Types

Both Q25 and Q28 test uncommon skills -- synthesizing positions for a compromise and inferring views from a specific proposal's wording. Neither lesson teaches the general methodology for these question types.

**Recommendation:** Add methodology callouts to 57-3 (compromise strategy) and 57-6 (expert viewpoint inference strategy).

### 5. No Module Wrap-Up or Synthesis

As with M55 and M56, the module ends after the final question without a synthesis lesson.

**Recommendation:** Consider appending a "Module Takeaways" section to 57-6 or adding a 57-7 synthesis lesson.
