# Module 46: Practice Passage 5 -- Cosmic Justice (Comparative) -- Website Content Improvement Report

**Scope:** The 9 website lessons (46-1 through 46-9) evaluated as a standalone course unit.
**Source:** PT-128-S-4-P-4 (Questions 20-27)
**Date:** 2026-04-01

---

## Executive Summary

Module 46 is a well-constructed comparative passage practice set with high-quality analyses. The passage pair (Cosmic Justice) is an excellent choice for teaching comparative RC -- both passages share a topic but diverge in their reasoning. The 8 question lessons cover a good spread of question types and all analyses are accurate. The primary improvement opportunities are:

1. **No "try it yourself" prompts before any analysis.** All 8 question lessons (46-2 through 46-9) jump directly from the question stem and answer choices into the breakdown. The student never has an opportunity to attempt the question independently.
2. **The passage analysis (46-1) is strong but could benefit from a strategy callout.** The "Head-to-Head" section is the best part of the lesson -- it should be more prominent and include a brief note on how to approach comparative passages generally.
3. **Passage is duplicated in every lesson.** The accordion passage block is identical across all 9 lessons. This is expected for the RC format but worth noting -- it means the passage analysis in 46-1 must stand fully on its own since students may skip re-reading.
4. **No approach tips or strategy guidance.** Unlike LR modules, there are no callouts explaining how to approach each question type (e.g., "For title-matching questions, find the thesis of each passage first").
5. **Answer choice statistics are included**, which is excellent for showing students which distractors are most dangerous.

## Lesson-by-Lesson Review

### Lesson 46-1: Passage Analysis (12 blocks)

**Role:** Full passage + structural breakdown for both passages.

**Strengths:**
- Clear paragraph-by-paragraph analysis for each passage.
- The "Head-to-Head: Inability vs. Illegitimacy" section precisely identifies the core difference between the passages -- this is the single most valuable teaching moment.
- The summary callout has clean bullet points for each passage's position and the core difference.
- The accordion for the passage text keeps the UI clean.

**Issues:**
- **No reading strategy guidance.** A brief callout on how to approach a comparative passage would be valuable -- e.g., "Read Passage A first and summarize its thesis before moving to Passage B" or "Look for agreement and disagreement points."
- **No question-type preview.** Students would benefit from knowing what types of questions to expect (e.g., "This set includes relationship questions, detail-finding, inference, and analogy questions").

**Suggested edits:**
- Add a callout before the passage analysis with a brief comparative passage strategy tip.
- Consider adding a "What to Expect" note listing the question types in this set.

## Structural Issues

### 1. No Active Practice Prompt Anywhere in the Module

The module is entirely passive. A student can read through all 9 lessons without ever being asked to attempt a question on their own. The question-card format shows the question and choices, but the analysis follows immediately with no pause for self-testing.

**Recommendation:** Add a "Try this" prompt before every worked analysis section.

### 2. No Question-Type Labels

Students cannot easily connect practice questions to the RC question taxonomy. Adding type labels would help students recognize patterns across passages and build metacognitive awareness.

**Recommendation:** Add a question-type label (from the Module 50 reference guide taxonomy) to each question lesson.

### 3. No Comparative Passage Strategy

This is the first comparative passage in the practice sequence (Module 42 covers this topic in theory). A brief strategy refresher in 46-1 would help students who jump directly to practice.

**Recommendation:** Add a strategy callout in 46-1 covering comparative passage reading order and the key task of identifying agreement/disagreement.
