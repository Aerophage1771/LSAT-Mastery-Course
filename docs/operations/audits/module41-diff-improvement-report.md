# Module 41: Weaken-Strengthen -- Website Content Improvement Report

**Scope:** The 2 website lessons (41-1 through 41-2) evaluated as a standalone course unit.
**Date:** 2026-04-01

---

## Executive Summary

Module 41 covers Weaken-Strengthen questions -- a question type that directly imports core Logical Reasoning skills into the Reading Comprehension section. At 78 appearances (~4%), this is a moderate-frequency type with high skill transfer value. The module is one of the stronger entries in the RC sequence:

1. **Zero practice questions** -- neither lesson includes a question-card or complete worked example. For a question type that mirrors LR skills, this is a missed opportunity to leverage students' existing knowledge.
2. **The dual-type coverage (Weaken + Strengthen) is well-handled** -- both 41-1 and 41-2 address both variants in a unified framework rather than splitting them into separate lessons. This is the right approach since the analytical method is identical, just reversed.
3. **Strong stem identification** -- the 41-1 breakdown clearly differentiates Weaken stems (weaken, undermine, cast doubt) from Strengthen stems (strengthen, support, lend credence) with color-coded badges.
4. **The "assume each answer choice is true" instruction is critical** -- both lessons emphasize this requirement, which is the single most important tactical insight for this question type.
5. **Good LR crossover note** -- the module correctly identifies the LR-to-RC skill transfer, which helps students who are stronger in LR leverage that advantage.

## Lesson-by-Lesson Review

### Lesson 41-1: Germaine's Guide to RC Weaken-Strengthen Questions (~15 blocks)

**Role:** Module introduction, concept overview, and stem identification guide.

**Strengths:**
- Clear opening that correctly frames the task: evaluating how new information affects a passage argument
- The four-item analytical framework (isolate argument, weaken patterns, strengthen patterns, logical impact focus) is comprehensive
- The explicit Weaken vs. Strengthen stem contrast with red and green badges is one of the clearest stem identification breakdowns in the RC modules
- Real stem language examples are varied and cover the full range of phrasing
- The Overarching Takeaway callout correctly reframes the student as a "logical critic" and emphasizes the Evidence → Conclusion structure
- The LR crossover note (78 appearances, ~4%) correctly positions this as a skill-transfer opportunity

**Issues:**
- **No practice question.** The Weaken/Strengthen framework is described but never tested.
- **No explicit comparison to LR Weaken/Strengthen.** Since the module notes the skill transfer, a brief table showing what is the same and what is different between LR and RC versions would be valuable. The key difference is that RC Weaken/Strengthen requires locating the argument in a longer passage before applying the same logical analysis.
- **The "assume each answer choice is true" instruction is mentioned only in passing.** This is the most important tactical insight and deserves a dedicated callout or emphasis.
- **No common wrong-answer patterns.** The module does not warn about the most frequent traps: answers that weaken a different argument, answers that go in the opposite direction, and answers with no logical impact.

**Suggested edits:**
- Promote "assume each answer choice is true" to a standalone callout with emphasis.
- Add a brief "RC vs. LR" comparison table: same skill (evaluate logical impact), different context (must locate the argument in a passage first).
- Add a "Common Traps" callout: (1) wrong argument, (2) opposite direction, (3) no impact.
- Add a brief exercise or stem identification practice.

### Lesson 41-2: Step-by-Step Approach (~14 blocks)

**Role:** The complete method for solving Weaken-Strengthen questions.

**Strengths:**
- Clean three-step method: Isolate the Argument (Conclusion + Evidence), Predict the Kind of Information Needed, Evaluate Each Answer's Logical Impact
- Step 1's Evidence → Conclusion structure decomposition is clear and includes a concrete example (tool marks → early humans were hunters)
- Step 2's prediction framework is well-differentiated for Weaken (attacks the link) vs. Strengthen (reinforces the link) with specific patterns (alternative explanation / ruling out alternatives)
- Step 3 correctly emphasizes assuming each answer is true and assessing its effect
- Two tie-breakers (Direct Impact Test, Scope Match Test) address the two most common decision points
- The Scope Match example (Cretaceous mammals vs. all animals throughout history) is concrete and memorable

**Issues:**
- **No question-card.** The method is explained well but never practiced.
- **No worked example.** The tool-marks example shows argument decomposition but does not walk through answer-choice evaluation.
- **The alternative-explanation pattern deserves more emphasis.** This is by far the most common pattern for both Weaken (providing an alternative) and Strengthen (ruling one out), and it gets only a bullet point. A dedicated example would help.
- **No "neither weaken nor strengthen" warning.** Some answer choices have no logical impact on the argument. Students need to be warned about these no-impact distractors.
- **Step 1 (Isolate the Argument) is uniquely challenging in RC.** Unlike LR where the entire stimulus is the argument, RC requires students to locate the relevant argument within a longer passage. A note about this added difficulty would help.

**Suggested edits:**
- Add a dedicated alternative-explanation example showing both Weaken and Strengthen versions of the same scenario.
- Add a question-card after the method section.
- Add a "no impact" warning for answer choices that discuss the topic without affecting the argument's logic.
- Add a note about the RC-specific challenge of locating the argument within a longer passage before applying the method.

## Structural Issues

### 1. No Practice Questions

Both lessons are pure instruction. Weaken-Strengthen questions mirror LR skills that many students already have, making practice especially high-value: students could confirm their LR skills transfer to RC with just one question-card.

**Recommendation:** Add at least one question-card to 41-2, ideally one that requires locating the argument within a passage to test the full RC-specific process.

### 2. Missing LR-to-RC Bridge

The module notes the LR crossover but does not provide an explicit comparison. Students transitioning from LR study to RC study would benefit from understanding what changes (context) and what stays the same (logical analysis).

**Recommendation:** Add a brief comparison note to 41-1.

### 3. Alternative Explanation Pattern Underweighted

The alternative-explanation pattern is the dominant pattern for both Weaken and Strengthen questions, but it receives only a bullet point in each lesson. It deserves prominent treatment.

**Recommendation:** Add a dedicated worked example showing the alternative-explanation pattern in both Weaken and Strengthen directions.
