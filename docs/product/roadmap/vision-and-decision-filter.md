# Vision And Decision Filter

**Purpose:** Define how LSAT Mastery Course roadmap work should be evaluated before it is prioritized.  
**Audience:** Product contributors deciding what is worth planning next.  
**Status:** active  
**Source of truth:** yes  
**Last reviewed:** 2026-03-20

## Decision Filter

1. Reduce question-integrity or content-drift risk
2. Improve study-loop clarity without inventing missing backend systems
3. Improve export, packaging, and validation reliability
4. Defer polish-only work unless it clearly supports one of the items above
