# Roadmap Bet Examples

**Purpose:** Provide template examples and active drafts of net-new, forward-looking product bets.
**Audience:** Product contributors deciding what to build next.
**Scope:** Differentiated curriculum expansion, pedagogy, assessment systems, exportable assets, licensing, and bridge-product bets.

This directory is strictly for high-value future additions to the LSAT Mastery Course. It should **not** include maintenance cleanup, pure bug-fix, or audit tasks unless they serve as explicit prerequisites for a larger strategic bet.
