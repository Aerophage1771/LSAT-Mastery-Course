# Roadmap Examples

**Purpose:** Provide concrete examples of forward-looking bets for curriculum expansion, product packaging, assessment systems, exportable assets, and higher-value learning surfaces.
**Audience:** Product contributors, maintainers, and agents generating new roadmap bets.
**Status:** active
**Source of truth:** yes

These examples illustrate how to propose net-new bets that could materially increase the product value of the course, moving beyond maintenance cleanup and pure bug fixes. They demonstrate the required structure for roadmap bets:

- User Problem
- Proposed Direction / Bet
- Why Now
- Dependencies
- Risks
- Likely Upside (Optional but recommended)
- First Practical Milestone
