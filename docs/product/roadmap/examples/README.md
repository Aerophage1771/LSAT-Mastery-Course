# Roadmap Bet Examples

**Purpose:** Provide example forward-looking roadmap bets for curriculum expansion, product packaging, assessment systems, exportable assets, and higher-value learning surfaces.
**Audience:** Maintainers, product contributors, and roadmap agents planning the course's future.
**Status:** reference
**Source of truth:** no (these are examples and proposals)

## Contents

This folder contains net-new, differentiated roadmap bets that materially increase the product value of the course. These do not duplicate existing maintenance, polish, or cleanup bets found in the root roadmap folder.

These files serve as templates and inspiration for future roadmap additions. Each file includes the user problem, proposed bet, why now, dependencies, risks, and a concrete first milestone.
