# Example Roadmap Bets

**Purpose:** Provide concrete examples of high-value, forward-looking roadmap bets for the LSAT Mastery Course. These serve as models for future curriculum expansion, assessment systems, and exportable assets.
**Audience:** Product contributors, content editors, and maintainers deciding what is worth planning next.
**Status:** active
**Source of truth:** yes
**Last reviewed:** 2026-04-16

## Current Landscape

- The current repo serves as a client-side SPA with no backend, relying on `localStorage` for state and exports for curriculum packaging.
- Future bets should focus on differentiated product value: curriculum expansion, pedagogy, licensing/B2B capability, offline reliability, and bridge products.
- Avoid treating pure maintenance tasks (cleanup, audit) as product bets, unless they unlock significant new user value.

## Example Bets in this Directory

- `27-institutional-lms-licensing-export.md`: B2B LMS licensing export capability
- `28-client-side-predictive-weakness-engine.md`: Client-side predictive weakness and adaptive assessment engine
- `29-printable-flashcard-asset-generator.md`: Anki-compatible CSV and printable flashcard generator

These files follow the required bet structure: User Problem, Proposed Bet, Why Now, Dependencies, Risks, and First Practical Milestone.
