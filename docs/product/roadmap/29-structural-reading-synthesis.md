# Structural Reading Synthesis

**Purpose:** Define the roadmap bet for synthesizing fragmented, overlapping RC structural modules into a unified whole.
**Audience:** Maintainers, content editors, and future agents planning course improvements.
**Status:** draft
**Source of truth:** yes
**Last reviewed:** 2026-04-20
**Related docs:** [README.md](./README.md), [vision-and-decision-filter.md](./vision-and-decision-filter.md), [../course-content-map.md](../course-content-map.md)

## Problem

- Content clustering and imbalanced lesson coverage: The RC curriculum currently has several thin, overlapping modules dealing with structural analysis (e.g., Module 28: Approach, Module 36: Paragraph Function, Module 37: Passage Function, Module 39: Structure).
- These tiny modules segment a single underlying skill—understanding macro-structure—into artificially separated buckets, creating an overly disjointed study experience that leaves value on the table.

## Proposed Direction

- Synthesize these fragmented modules into a single, robust "Macro-Structure & Method" RC unit.
- Teach structural reading, approach, and paragraph/passage function as interconnected skills rather than isolated question types.

## Why Now

- The course-content-map reveals that many of these modules contain only one or two lessons.
- Consolidating them now will clean up the route map, eliminate redundancy, and create a more compelling and substantial lesson block that justifies its place in the curriculum.

## Likely Upside

- Creates a stronger offer by presenting a unified theory of structural reading, which is arguably the most critical RC skill.
- Better operator leverage: maintaining one comprehensive structural unit is easier than maintaining four overlapping micro-modules.
- Dramatically improves module landing page quality and perceived value for these lessons.

## First Practical Milestone

- Draft the unified syllabus for the new "Macro-Structure & Method" unit, mapping the existing content from modules 28, 36, 37, and 39 into a logical, sequential lesson flow.
- Re-route the course catalog logic to point the old module IDs to the new unified structure.
