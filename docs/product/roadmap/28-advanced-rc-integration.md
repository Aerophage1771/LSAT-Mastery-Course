# Advanced RC Integration

**Purpose:** Define the roadmap bet for integrating the isolated "Advanced RC" passages into the core RC concept modules.
**Audience:** Maintainers, content editors, and future agents planning course improvements.
**Status:** draft
**Source of truth:** yes
**Last reviewed:** 2026-04-20
**Related docs:** [README.md](./README.md), [vision-and-decision-filter.md](./vision-and-decision-filter.md), [../course-content-map.md](../course-content-map.md)

## Problem

- Structural holes that weaken the course map: Modules 51-57 currently house "Advanced RC" passages as isolated, standalone modules.
- This structure differs from Logical Reasoning (LR), where advanced concepts and difficult questions are integrated directly into the relevant core concept modules as "Advanced Concepts" lessons. This creates a disjointed learning experience for advanced RC.

## Proposed Direction

- Dissolve the standalone Advanced RC modules (51-57).
- Integrate their passages and associated question-breakdown lessons into the relevant core RC concept modules as "Advanced Lessons" (e.g., "Advanced Lesson: Difficult Inference Questions").
- Mirror the successful structural pattern already established in LR.

## Why Now

- The current structure creates an artificial separation between learning the core skill and applying it to high-difficulty material.
- Re-architecting this now prevents further divergence between the LR and RC course structures as new content is added.

## Likely Upside

- Creates a more cohesive, unified learning path for RC.
- Strengthens the internal coherence of the course catalog.
- Simplifies routing and curriculum maintenance by reducing the number of standalone, disconnected modules.

## First Practical Milestone

- Map the specific passages and questions from Modules 51-57 to their most appropriate target core RC modules.
- Execute the migration for one advanced passage as a proof-of-concept, verifying that the catalog logic and downstream exports handle the change cleanly.
