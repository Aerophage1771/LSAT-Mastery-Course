# LR-RC Synthesis Modules

**Purpose:** Define the roadmap bet for building curriculum bridges between Logical Reasoning and Reading Comprehension.
**Audience:** Maintainers, content editors, and future agents planning course improvements.
**Status:** active
**Source of truth:** yes
**Last reviewed:** 2026-03-29
**Related docs:** [README.md](./README.md), [vision-and-decision-filter.md](./vision-and-decision-filter.md), [../../technical/content-and-validation.md](../../technical/content-and-validation.md)

## User Problem

- The course currently treats Logical Reasoning and Reading Comprehension as completely separate silos, with no explicit linkage in the curriculum map.
- Users miss the opportunity to apply advanced LR argument analysis skills (e.g., Argument Part, Method of Reasoning) to the complex, multi-paragraph structures found in RC.
- The lack of cross-disciplinary training leaves value on the table and weakens the internal coherence of the overall methodology.

## Proposed Direction (The Bet)

- Develop dedicated "Synthesis" modules that explicitly bridge LR argument skills with RC passage analysis.
- Create lessons that teach users how to map LR structural concepts (premises, intermediate conclusions, opposing viewpoints, concessions) onto full-length RC passages.
- Position these modules as an advanced tier that unlocks after core competencies in both LR and RC are established.

## Why Now

- Both the LR and RC core curriculums are mature enough to support advanced, cross-disciplinary integration.
- This bet offers net-new value that differentiates the course methodology without requiring any new backend infrastructure.
- It addresses a structural hole in the curriculum by explicitly teaching the connections that top scorers make intuitively.

## Likely Upside

- Strengthens internal coherence by reinforcing a unified logic-based approach across different section types.
- Provides a high-value, advanced learning tier that can be marketed as a distinct pedagogical advantage.
- Increases overall operator leverage by reusing core concepts across multiple contexts.

## First Practical Milestone

- Create a pilot "Synthesis: Argument Structure in RC" module featuring 3-4 lessons that map the `Argument Part` LR framework onto a complex, multi-viewpoint RC passage.
