# Archive

**Purpose:** Explain how historical plans, brainstorms, and superseded docs are stored.  
**Audience:** Contributors who need old context without confusing it for live truth.  
**Status:** active  
**Source of truth:** yes  
**Last reviewed:** 2026-03-11  
**Related docs:** [../README.md](../README.md), [../product/roadmap/README.md](../product/roadmap/README.md), [../operations/audits/README.md](../operations/audits/README.md)

## What Belongs Here

- dated plans
- brainstorms
- superseded proposals
- old snapshots that still provide historical context

## Archive Rules

- Archived docs are context only, not live truth.
- Historical docs should say what replaced them.
- Active docs should not point readers here unless they need historical background.
- If a source-of-truth doc moves or is retired, update the active replacement first and then archive the older material.
